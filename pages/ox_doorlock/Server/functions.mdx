# 関数

指定されたIDのドアのデータを取得します。データベースエントリのIDと一致します。

## getDoor

```lua
exports.ox_doorlock:getDoor(doorId)
```

指定された `id` のドアのデータを取得します。データベースエントリのIDと一致します。

- id: `number`

戻り値:

- door: `table`

## getDoorFromName

```lua
exports.ox_doorlock:getDoorFromName(name)
```

指定された `name` のドアのデータを取得します。データベースエントリの名前と一致します。

- name: `string`

戻り値:

- door: `table`

## editDoor

```lua
exports.ox_doorlock:editDoor(doorId, data)
```

指定されたdoorIdの構成を編集します。

- doorId: `number`
- data: `table`

## setDoorState

```lua
exports.ox_doorlock:setDoorState(doorId, state)
```

stateが `true` または `1` の場合、指定されたdoorIdのドアをロック済みとして設定します。

- doorId: `number`
- state: `0` または `1` または `boolean`
