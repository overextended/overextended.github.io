# イベント

## ハンドラー

これらのイベントは、他のスクリプトによってトリガーされるべきでは**ありません**。

### ox_doorlock:stateChanged

ドアの状態が更新されたときにトリガーされます。

```lua
AddEventHandler('ox_doorlock:stateChanged', function(source, doorId, state, usedItem) end)
```

- source: `number` または `nil`
- doorId: `number`
- state: `boolean`
- usedItem: `string` または `false` または `nil`
