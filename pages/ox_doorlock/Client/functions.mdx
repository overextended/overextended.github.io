# 関数

## pickClosestDoor

最も近いドアのロックを解除しようとします。サーバー側のチェックに依存し、失敗する可能性があります。

```lua
exports.ox_doorlock:pickClosestDoor()
```

## useClosestDoor

最も近いドアと対話します。サーバー側のチェックに依存し、失敗する可能性があります。

```lua
exports.ox_doorlock:useClosestDoor()
```

## getClosestDoor

最も近いドアを返します。

```lua
exports.ox_doorlock:getClosestDoor()
```

戻り値:

- door: `number`
