# クラフト

クラフト場所、アイテム、およびそれらの材料は[data/crafting.lua](https://github.com/overextended/ox_inventory/blob/main/data/crafting.lua)で定義されています。

## クラフト定義

```lua
{
    items = {
        {
            name = 'lockpick',
            ingredients = {
              garbage = 3,
              WEAPON_HAMMER = 0.1
            },
            duration = 5000,
            count = 3,
            metadata = { durability = 20 }
        },
        {
            name = 'garbage',
            ingredients = {
              cola = 1
            },
            metadata = { description = 'An empty soda can.', weight = 20, image = 'trash_can' }
        },
    },
    points = {
      vec3(-1147.083008, -2002.662109, 13.180260),
    },
    zones = {
        {
          label = 'Open Crafting Bench',
          icon = 'fa-solid fa-wrench',
          coords = vec3(-1146.2, -2002.05, 13.2),
          size = vec3(3.8, 1.05, 0.15),
          distance = 1.5,
          rotation = 315.0,
        },
    },
    blip = { id = 566, colour = 31, scale = 0.8 },
    name = 'crafting-bench',
},
```

- items: `table`
  - name: `string`
  - ingredients: `table`
    - アイテムの材料はアイテムのツールチップに表示されます。
    - アイテム名と消費数のキーと値のペア
      - key - アイテム名。
      - value - 1以上の場合は消費数、1未満で0より大きい場合は耐久性消費量、
        0に設定されている場合はアイテムは必要ですが消費されません。
  - duration: `number`
    - クラフト時間（ミリ秒）。
  - count: `number` または `table` (最小、最大)
    - クラフト時に受け取るアイテムの量。
    - テーブルに設定する場合、最初の数値は最小数、2番目の数値は最大数である必要があり、クラフトされたアイテムをプレイヤーに追加するためにこれらの2つの数値の間の乱数を生成します。
  - metadata: `table`
    - クラフトされるアイテムに適用されるメタデータ。
- points: `vector3[]`
  - クラフトインベントリを開く操作場所。
- groups: `table`
  - クラフト場所にアクセスするためのジョブ名と最小グレードのキーと値のペア。
  - `{["police"] = 0, ["ambulance"] = 2}`
- zones: `table`
  - ox_targetに使用されるox_libターゲティングゾーン。
  - label: `string`
  - icon: `string`
  - coords: `vector3`
  - size: `vector3`
  - distance: `number`
  - rotation: `number`
- blip: `table`
  - id: `number`
    - ブリップスプライト番号。
  - colour: `number`
  - scale: `number`
- name: `string`
  - openInventoryエクスポートを使用するときにIDとして使用できるオプションの名前。
