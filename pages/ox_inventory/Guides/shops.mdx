# ショップの作成

組み込みショップは[data/shops.lua](https://github.com/overextended/ox_inventory/blob/main/data/shops.lua)で定義されており、組み込みのマーカーまたはゾーンサポートの恩恵を受けるためにここに追加できます。

## ショップ定義

```lua
{
    General = {
        name = 'ショップ',
        blip = {
          id = 59,
          colour = 69,
          scale = 0.8
        },
        inventory = {
            { name = 'burger', price = 10 },
            { name = 'water', price = 10 },
            { name = 'cola', price = 10 },
        },
        locations = {
            vec3(25.7, -1347.3, 29.49),
        },
        targets = {
            -- BoxZoneを使用したショップ
            {
                loc = vec3(25.06, -1347.32, 29.5),
                length = 0.7,
                width = 0.5,
                heading = 0.0,
                minZ = 29.5,
                maxZ = 29.9,
                distance = 1.5
            },
            -- 歩行者を使用したショップ
            {
                ped = `mp_m_shopkeep_01`,
                scenario = 'WORLD_HUMAN_AA_COFFEE',
                loc = vec3(24.407, -1347.283, 28.497),
                heading = 270.311,
            },
        }
    }
}
```

- name: `string`
  - ショップが開いているときに表示されるラベル。
- blip?: `table`
  - 指定された設定でブリップを作成します。ブリップを作成しない場合は未定義のままにします。
  - id: `number`
  - colour: `number`
  - scale: `number`
- groups?: `table`
  - ショップにアクセスするためのジョブ名と最小グレードのキーと値のペア。
    - `{["police"] = 0, ["ambulance"] = 2}`
- inventory: `table`
  - name: `string`
  - price: `number`
  - currency?: `string`
    - 通貨として使用されるアイテム。
  - count?: `number`
    - 在庫にあるアイテムの量。
  - license?: `string`
    - アイテムの購入に必要なライセンス。
  - metadata?: `table`
  - grade?: `number` | `number[]`
    - アイテムの購入に必要な最小グレード。
- locations?: `vector3[]`
  - マーカーを使用して、ショップアーキタイプのユニークなインスタンスを作成する座標の配列。
- targets?: `table[]`
  - 歩行者またはBoxZone（PolyZoneデータ構造）を使用して、ショップアーキタイプのユニークなインスタンスを作成するターゲット設定の配列。
- model?: `number[]`
  - ショップを開くためにターゲットにできるモデルの配列。自動販売機に使用されます。

ターゲットとモデルは、ox_targetなどのターゲティングリソースを使用する場合にのみ使用できます。

## ランタイム中の登録

ショップはサーバーで `exports.ox_inventory:RegisterShop` を使用して追加できますが、クライアント専用機能は利用できません。

- ブリップ、マーカー、ゾーンは作成されません。
- アーキタイプを使用して各ショップを定義するには、「targets」ではなく「locations」を使用する必要があります。

### 例

```lua
exports.ox_inventory:RegisterShop('TestShop', {
    name = 'テストショップ',
    inventory = {
        { name = 'burger', price = 10 },
        { name = 'water', price = 10 },
        { name = 'cola', price = 10 },
    },
    locations = {
        vec3(223.832962, -792.619751, 30.695190),
    },
    groups = {
        police = 0
    },
})
```
