# メタデータ

アイテムメタデータは、単一のアイテムから複数の異なるアイテムを作成するために使用できる非常に強力なツールです。

このガイドではポケモンカードを例として使用しますが、`garbage`アイテムを使用してインベントリに既に統合されている例を見つけることができます。

## ベースアイテムの作成

まず、メタデータを適用するために使用するベースアイテムを作成する必要があります。

```lua filename="/data/items.lua"
['pokemon_card'] = {
    label = 'ポケモンカード',
    weight = 10,
    consume = 0,
    server = {
        export = 'pokemon.pokemon_card'
    }
}
```

この場合、すべてのカードの重量を同じにするため、ラベルと重量も定義しますが、
すべてを同じ重量にしたくない場合は、省略してメタデータを通じて重量を適用できます。

また、`pokemon`リソースの`pokemon_card`エクスポートを呼び出すことで、アイテムを使用可能にします。

```lua filename="pokemon/server.lua"
exports('pokemon_card', function(event, item, inventory, slot, data)
    if event == 'usingItem' then
        local itemSlot = exports.ox_inventory:GetSlot(inventory.id, slot)
        print(json.encode(itemSlot.metadata, {indent=true}))
    end
end)
```

## 特別なメタデータプロパティ

任意のメタデータプロパティを任意の値で定義できますが、特別なユースケースを持つメタデータプロパティがいくつかあります。

これらのプロパティは次のとおりです。

- label: `string`
  - アイテムの表示名
- weight: `number`
  - アイテムの重量
- description: `string`
  - ツールチップに表示されるアイテムの説明
- image: `string`
  - アイテムが使用する画像パス内の画像
- imageurl: `string`
  - アイテムが使用する画像へのURL
- type: `any`
  - ツールチップの右上に表示されるアイテムタイプ

これらのプロパティを使用して、以前に作成した`pokemon_card`アイテムからポケモンカードを作成します。

## メタデータアイテムの作成

[createItem](../Functions/Server/Hooks#createitem)を使用してフックを定義し、ショップに追加することで、メタデータアイテムを簡単に作成できます。

```lua filename="/data/shops.lua"
inventory = {
    {name = 'pokemon_card', price = 300, metadata = {
        label = 'リザードン',
        description = 'リザードンの炎は、過酷な戦いを経験した場合、より熱く燃えると言われています。',
        image = 'panties',
        type = '炎',
        hp = 78,
        attack = 84,
        defense = 78
    }}
}
```

```lua filename="pokemon/server.lua"
local pokemonMetadata = {
    charizard = {
        label = 'リザードン',
        description = 'リザードンの炎は、過酷な戦いを経験した場合、より熱く燃えると言われています。',
        image = 'panties',
        type = '炎',
        hp = 78,
        attack = 84,
        defense = 78
    }
}

local hookId = exports.ox_inventory:registerHook('createItem', function(payload)
    local pokemon = pokemonMetadata[payload.metadata.type]
    if not pokemon then return end
    return pokemon
end, {
    itemFilter = {
        pokemon_card = true
    }
})
```

上記のように、アイテムが使用可能になると、メタデータプロパティはすべてそこにあり、スロットからアクセスできます。

## カスタムメタデータプロパティの表示

文字列連結を使用して説明に追加するか、[displayMetadata](../Functions/Client#displaymetadata)_クライアント_関数を使用して、リザードンカードに設定したカスタムメタデータを表示できます。

```lua
exports.ox_inventory:displayMetadata({
    hp = 'HP',
    attack = '攻撃',
    defense = '防御'
})
```
