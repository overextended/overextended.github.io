# 共有

## lib.getClosestVehicle

座標のセットに最も近い車両の車両ハンドルと座標を取得します。

```lua
lib.getClosestVehicle(coords, maxDistance, includePlayerVehicle)
```

- coords: `vector3`
  - チェックする座標。
- maxDistance?: `number`
  - チェックする最大距離。
  - デフォルト: `2.0`
- includePlayerVehicle?: `boolean`
  - プレイヤーの現在の車両を含めるかどうか。サーバーでは無視されます。
  - デフォルト: `false`

戻り値:

- vehicle?: `number`
- vehicleCoords?: `vector3`
