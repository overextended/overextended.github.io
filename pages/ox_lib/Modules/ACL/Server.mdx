# サーバー

組み込みACLシステムのラッパー。lib.addCommandおよびox_groups権限を処理します。
詳細については、[Basic Aces & Principalsの概要/ガイド](https://forum.cfx.re/t/basic-aces-principals-overview-guide/90917)を参照してください。

## lib.addAce

プリンシパルにace権限を割り当てます。3番目のパラメータはデフォルトで「allow」になり、`false`を渡すと権限が「deny」に設定されます。

<Tabs items={["Lua", "JS"]}>
    <Tab>
        ```lua
        lib.addAce(principal, ace, allow)

        lib.addAce('group.admin', 'command.say')
        ```
    </Tab>
    <Tab>
        ```ts
        import lib from '@overextended/ox_lib/server'

        lib.addAce(principal, ace, allow)

        lib.addAce('group.admin', 'command.say')
        ```
    </Tab>
</Tabs>

- principal: `string`
- ace: `string`
- allow: `boolean`

## lib.removeAce

プリンシパルからace権限を削除します。3番目のパラメータはデフォルトで「allow」になり、`false`を渡すと権限が「deny」に設定されます。

<Tabs items={["Lua", "JS"]}>
    <Tab>
        ```lua
        lib.removeAce(principal, ace, allow)

        lib.removeAce('group.admin', 'command.say')
        ```
    </Tab>
    <Tab>
        ```ts
        import lib from '@overextended/ox_lib/server'

        lib.removeAce(principal, ace, allow)

        lib.removeAce('group.admin', 'command.say')
        ```
    </Tab>
</Tabs>

- principal: `string`
- ace: `string`
- allow: `boolean`

## lib.addPrincipal

プリンシパルを親プリンシパルに割り当てます。子は親から権限を継承します。

<Tabs items={["Lua", "JS"]}>
    <Tab>
        ```lua
        lib.addPrincipal(child, parent)

        lib.addPrincipal('player.1', 'group.moderator')
        ```
    </Tab>
    <Tab>
        ```ts
        import lib from '@overextended/ox_lib/server'

        lib.addPrincipal(child, parent)

        lib.addPrincipal('player.1', 'group.moderator')
        ```
    </Tab>
</Tabs>

- child: `string`
- parent: `string`

## lib.removePrincipal

親プリンシパルからプリンシパルを削除します。

<Tabs items={["Lua", "JS"]}>
