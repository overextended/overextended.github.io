# クライアント

[共有](../Shared)値に加えて、クライアントで利用可能な値とキャッシュ機能。

## デフォルト値

- ped: `number`
  - プレイヤーエンティティID
- playerId: `number`
  - プレイヤーID
- serverId: `number`
  - プレイヤーサーバーID
- weapon: `number` または `false`
  - 現在の武器ハッシュ
- vehicle: `number` または `false`
  - 車両エンティティID
- seat: `number` または `false`
  - 車両シートインデックス
- coords: `vector3`
  - 現在のプレイヤー座標
  - `zones` または `points` を使用している場合にのみ入力されます
  - `lib.onCache` でリッスンできません
- mount: `number` または `false` (RedMのみ)
  - マウントエンティティID

## lib.onCache

キャッシュされた値が更新されたときにトリガーされるイベントハンドラーを登録します。

<Tabs items={["Lua", "JS"]}>
  <Tab>
    ```lua
    lib.onCache(key, function(value, oldValue) end)
    ```

    - key: `string`
      - `ped`
      - `vehicle`
      - `seat`
      - `weapon`
      - `mount` (RedMのみ)
    - value: `any`

    ```lua
    lib.onCache('vehicle', function(value, oldValue)
        print('古い車両:', cache.vehicle)
        print('新しい車両:', value)
    end)
    ```
  </Tab>
  <Tab>
    ```ts
    import { onCache } from '@overextended/ox_lib/client';

    onCache(key, (value) => {});
    ```

    - key: `string`
      - `ped`
      - `vehicle`
      - `seat`
      - `weapon`
      - `mount` (RedMのみ)
    - value: `any`

    ```ts
    import { cache, onCache } from '@overextended/ox_lib/client';

    lib.onCache('vehicle', (value) => {
      console.log('古い車両:', cache.vehicle);
      console.log('新しい車両:', value);
    });
    ```
  </Tab>
</Tabs>
