# 共有

## lib.array

Arrayクラスのプロトタイプとして使用されるテーブル。関数はクラスメソッドまたは標準関数として使用できます。

### lib.array:new

指定された要素を含むArrayのインスタンスを構築します。

- ...: `any`

```lua
lib.array:new(...)
```

**戻り値:**

- arr: `Array`

### lib.array.from

反復可能な値から新しい配列を作成します。

- iter: `function` | `table` | `string`
  - string.gmatchなどのイテレータ関数を使用して配列を構築できます。

```lua
lib.array.from(iter)
```

**戻り値:**

- arr: `Array`

### lib.array.at

指定されたインデックスの要素を返します。負の数は配列の末尾から逆方向にカウントされます。

- index: `integer`

```lua
lib.array.at(index)
```

**戻り値:**

- element: `unknown`

### lib.array.merge

2つ以上の配列の要素を新しい配列に結合します。

- ...: `Array`

```lua
lib.array.merge(...)
```

**戻り値:**

- Array

### lib.array.every

配列内のすべての要素が、指定されたテスト関数に合格するかどうかをテストします。

- arr: `Array`
- testFn: `function(element: unknown): boolean`

```lua
lib.array.every(arr, testFn)
```

**戻り値:**

- success: `boolean`

### lib.array.fill

範囲内のすべての要素を指定された値に設定し、変更された配列を返します。

- arr: `Array`
- value: `any`
- start?: `integer`
- endIndex?: `integer`

```lua
lib.array.fill(arr, value, start, endIndex)
```

**戻り値:**

- arr: `Array`

### lib.array.filter

指定されたテスト関数に合格する配列の要素を含む新しい配列を作成します。

- arr: `Array`
- testFn: `function(element: unknown): boolean`

```lua
lib.array.filter(arr, testFn)
```

**戻り値:**

- newArr: `Array`

### lib.array.find

指定されたテスト関数に合格する配列の最初の要素を返します。

- arr: `Array`
- testFn: `function(element: unknown): boolean`
- reverse?: `boolean`
  - 配列を逆順に反復処理します。

```lua
lib.array.find(arr, testFn, reverse)
```

**戻り値:**

- element: `unknown`

### lib.array.findIndex

指定されたテスト関数に合格する配列の最初の要素のインデックスを返します。

- arr: `Array`
- testFn: `function(element: unknown): boolean`
- reverse?: `boolean`
  - 配列を逆順に反復処理します。

```lua
lib.array.findIndex(arr, testFn, reverse)
```

**戻り値:**

- index: `integer`

### lib.array.indexOf

指定された値に一致する配列の最初の要素のインデックスを返します。

- arr: `Array`
- value: `any`
- reverse?: `boolean`
  - 配列を逆順に反復処理します。

```lua
lib.array.indexOf(arr, value, reverse)
```

**戻り値:**

- index: `integer`

### lib.array.forEach

配列内の各要素に対して指定された関数を実行します。

- arr: `Array`
- cb: `function(element: unknown)`

```lua
lib.array.forEach(arr, cb, reverse)
```

**戻り値:**

- index: `integer`

### lib.array.join

配列のすべての要素を文字列に連結し、コンマまたは指定された区切り文字で区切ります。

- arr: `Array`
- seperator?: `string`

```lua
lib.array.join(arr, seperator)
```

**戻り値:**

- str: `string`

### lib.array.map

配列内の各要素に対して指定された関数を呼び出した結果を含む新しい配列を作成します。

- fn: `function(element: unknown, index: integer, arr: Array): unknown`

```lua
lib.array.map(arr, fn)
```

**戻り値:**

- newArr: `Array`

### lib.array.pop

配列から最後の要素を削除し、その値を返します。

- arr: `Array`

```lua
lib.array.pop(arr)
```

**戻り値:**

- element: `unknown`

### lib.array.push

指定された要素を配列の末尾に追加し、新しい配列の長さを返します。

- arr: `Array`
- ...: `any`

```lua
lib.array.push(arr, ...)
```

**戻り値:**

- length: `integer`

### lib.array.shift

配列から最初の要素を削除し、その値を返します。

- arr: `Array`

```lua
lib.array.shift(arr)
```

**戻り値:**

- element: `unknown`

### lib.array.slice

配列の一部を浅いコピーとして新しい配列に作成します。

- arr: `Array`
- start?: `integer`
- finish?: `integer`

```lua
lib.array.slice(arr, start, finish)
```

**戻り値:**

- newArr: `Array`

### lib.array.toReversed

指定された配列から要素の順序を逆にした新しい配列を作成します。

- arr: `Array`

```lua
lib.array.toReversed(arr)
```

**戻り値:**

- newArr: `Array`

### lib.array.unshift

新しい要素を配列の先頭に挿入し、新しい配列の長さを返します。

- arr: `Array`
- ...: `any`

```lua
lib.array.unshift(arr, ...)
```

**戻り値:**

- length: `integer`

### lib.array.reduce

「reducer」関数は配列内のすべての要素に適用され、前の結果がアキュムレータとして機能します。
初期値が指定されている場合、最初のインデックスのアキュムレータとして使用されます。それ以外の場合、反復は2番目のインデックスから開始され、最初のインデックスがアキュムレータになります。

- arr: `Array`
- reducer: `function(accumulator: unknown, element: unknown, index?: integer)`
- initialValue?: `any`

```lua
lib.array.reduce(arr, reducer)
```

**戻り値:**

- accumulator: `unknown`

### lib.array.reverse

配列内の要素の順序を逆にします。

- arr: `Array`

```lua
lib.array.reverse(arr)
```

**戻り値:**

- arr: `Array`

### lib.isArray

指定されたテーブルがArrayのインスタンスであるか、配列のようなテーブルであるかを判断します。

- tbl: `table`

```lua
lib.array.isArray(tbl)
```

**戻り値:**

- isArray: `boolean`
