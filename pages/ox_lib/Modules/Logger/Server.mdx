# サーバー

## lib.logger

```lua
lib.logger(source, event, message, ...)
```

- source: `number` または `string`
  - できればアクティブなプレイヤーID、それ以外の場合は識別子、またはそれがどこから来たか。
- event: `string`
  - ログイベントの名前（つまり、トリガーイベントまたは説明）
- message: `string`
  - ログの内容。
- vararg: `string`
  - 追加の引数は、追加のフィルタリングと検索のためにタグに変換されます。

<br />
<u>**Example**</u>

```lua
local vehicle = Ox.CreateVehicle(false, `sultanrs`, vector4(-56.479122, -1116.870362, 26.432250, 0.000030517578))
lib.logger(-1, 'CreateVehicle', json.encode(vehicle))
```

## Datadog

<Steps>
  ### [Datadog](https://www.datadoghq.com/)アカウントを作成します
  14日間の無料トライアルが提供されます。それ以外の場合は、[価格](https://www.datadoghq.com/pricing/)ガイドを参照してください。
  無料アカウントには制限がありますが、ログは14日間保持されます。

  ### [APIキー](https://app.datadoghq.com/organization-settings/api-keys)を作成します

  これは、Datadog組織にログを送信するために使用されるUUIDです。

  ### 設定

  次のconvarを使用して、APIキーと[Datadogサイト](https://docs.datadoghq.com/getting_started/site/)を設定します。

  ```bash
  set datadog:key "yourapikey"
  set datadog:site "datadoghq.com"
  ```
</Steps>

## Grafana Loki

Lokiは、Prometheusに触発された、水平方向にスケーラブルで可用性が高く、マルチテナントのログ集約システムです。
非常に費用対効果が高く、操作が簡単になるように設計されています。

<Tabs items={["クラウド", "セルフホスト"]}>
  <Tab>
    - いくつかの制限付きの無料ティアがあります
    - 管理が簡単
    - セットアップが簡単
    - 主にGrafanaによって管理されています

    <Steps>
      ### [Grafana](https://grafana.com/)アカウントを作成します
      ### [マイアカウント](https://grafana.com/auth/sign-in/?plcmt=top-nav&cta=myaccount)に移動します
      ### Lokiのセットアップ
      アカウントパネルでLokiセクションを見つけ、「詳細」をクリックします。
      パスワードを生成し、APIキー、および`user`と`url`を保存します。
    </Steps>
  </Tab>
  <Tab>
    Grafanaインスタンスをセットアップするには、dockerと、コンテナまたはkubernetesに関する知識が必要です。
    Grafanaスタック（最小要件はGrafanaとGrafana Loki）をセットアップするためのガイドを見つけて、それに従ってください。
    
    完了したら、認証を設定し、エンドポイントを保護するユーザー名とパスワードを使用します。

    - デフォルトでは、セルフホストLokiインスタンスは認証レイヤーを提供せず、NGINX基本認証やCloudflare Accessなどの外部認証レイヤーが必要になります。
  </Tab>
</Tabs>

  ### 設定
  次のconvarを使用して、ロギングサービス、エンドポイント、および認証の詳細を設定します。

  ```bash
  set ox:logger "loki"
  set loki:user "<insert your user>"
  set loki:password "<insert the api key or password>"
  set loki:endpoint "<insert loki url without http:// or https://>"
  ```


## Fivemanage
[Fivemanage](https://fivemanage.com/?ref=overextended) は、ロギングを提供するゲームサーバー向けの使いやすいオールインワンSaaSです。

<Steps>
    ### アカウントを作成
    [Fivemanage](https://fivemanage.com/?ref=overextended) でサインアップします。
    ### `Hobby` または `Pro` プランがあることを確認してください。
    サブスクリプションをまだお持ちでない場合は、14日間の無料トライアルをご利用いただけます。詳細については、[こちら](https://fivemanage.com/profile/billing?ref=overextended) および [価格](https://fivemanage.com/pricing?ref=overextended) ページをご覧ください。
    ### 新しいトークンを作成
    タイプ `Logs` の新しいトークンを作成する必要があります。
    ### 設定
  ```bash
  set ox:logger "fivemanage"
  set fivemanage:key "YOUR_API_KEY"
  ```
</Steps>
