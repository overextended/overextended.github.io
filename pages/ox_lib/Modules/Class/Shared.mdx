# 共有

## クラス

クラスは、設定された属性と動作を持つオブジェクトを作成するためのテンプレートです。

- Luaは通常オブジェクト指向プログラミングをサポートしていませんが、[メタテーブル](https://www.lua.org/manual/5.4/manual.html#2.4)を使用してシミュレートすることが可能です。
- 私たちのクラスの実装は、[JavaScript](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Classes)に大まかに基づいています。

### 継承

クラスは、インスタンスが親クラスから継承するのと同様に、他のクラスから属性と動作を継承できます。

クラス宣言で「スーパー」クラスを指定すると、派生クラスまたは「サブ」クラスが作成されます。このサブクラスは、親からフィールドとメソッドの両方を継承します。この継承は、派生クラスの複数のレベルにまたがり、クラス属性をクラス階層全体で共有できるようにします。

### カプセル化

カプセル化とは、データ（属性）とメソッド（動作）をクラス内にバンドルすることです。

この方法は、関連する機能を整理し、クラスの内部動作への外部からの干渉を防ぐのに役立ちます。これらの要素をカプセル化することにより、モジュール性を促進し、データの整合性を維持し、プログラム内でデータがどのようにアクセスおよび操作されるかを制御します。

## OxClass

OxClassは、`lib.class`で作成されたすべてのクラスに使用されるブループリントです。すべてのクラスで共有されるいくつかの基本的な属性と動作を提供します。

### 属性

- __name: `string`
  - クラスの名前を表す保護されたフィールド。
- constructor?: `function`
  - クラスから継承する新しいオブジェクトをインスタンス化するために使用される保護されたメソッド。
  - `class:constructor()` が存在する場合、`class:new(...)` によって呼び出されます。
- private?: `table`
  - クラス内のすべてのプライベートデータを保持する保護されたフィールド。
  - プライベートデータはクラスメソッドの外部からはアクセスできず、セキュリティとデータの整合性を強制します。
  - `constructor` で常に定義されますが、プライベートプロパティが設定されていない場合は明示的に定義する必要があります。
- super?: `class`
  - スーパークラスのコンストラクタを参照する保護されたメソッド。
  - クラスをインスタンス化するときにクラスコンストラクタ内で呼び出す必要があります。

## lib.class

この関数を使用すると、オプションで親または「スーパー」クラスから継承する新しいクラスを定義できます。

```lua
local MyClass = lib.class(name, super)
```

### Parameters

- name: `string`
- super?: `table`
  - 新しいクラスをスーパークラスのサブセットとして設定します。

### Returns
- `table`
  - 新しく作成されたクラス。

### Example

```lua
---@class Person : OxClass
---@field name string
local Person = lib.class('Person')

function Person:constructor(name)
    print('Personコンストラクタを呼び出しています：', name)
    self.name = name
end

-- Professorクラス（Personを拡張）

---@class Professor : Person
---@field teaches string
local Professor = lib.class('Professor', Person)

function Professor:constructor(name, teaches)
    print('Professorコンストラクタを呼び出しています：', name)
    self:super(name)
    self.teaches = teaches
end

function Professor:introduceSelf()
    print(("私の名前は%sで、%sの教授になります。"):format(self.name, self.teaches))
end

function Professor:grade(paper)
    local grade = math.random(1, 4)
    print(grade)
end

CreateThread(function()
    local walter = Professor:new('Walter', '化学')

    walter:introduceSelf()
    walter:grade('私の論文')
end)


-- Studentクラス（Personを拡張）

---@class Student : Person
---@field private private { year: number }
local Student = lib.class('Student', Person)

---@param name string
---@param year number
function Student:constructor(name, year)
    print('Studentコンストラクタを呼び出しています：', name)
    self:super(name)
    self.private.year = year
end

function Student:introduceSelf()
    print(("こんにちは！私は%sで、%s年生です。"):format(self.name, self.private.year))
end

---@param year number
function Student:setYear(year)
    self.private.year = year
end

CreateThread(function()
    local jesse = Student:new('Jesse', 2)

    jesse:introduceSelf() -- こんにちは！私はJesseで、2年生です。
    jesse:setYear(3)
    jesse:introduceSelf() -- こんにちは！私はJesseで、3年生です。
    print(jesse.private.year) -- nil
    print(getmetatable(jesse.private)) -- private
    jesse.private.year = 4 -- error
end)
```
