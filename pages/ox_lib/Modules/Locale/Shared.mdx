サーバーが優先言語を設定し、モジュールを使用する任意のリソースでロケールファイルの読み込みを試行できるようにします。
ロケールファイルは[ISO言語コード](http://www.lingoes.net/en/translator/langcode.htm)を使用し、`./locales/langcode.json` として保存する必要があります。

## セットアップ

優先言語を英語から変更するには、convarをserver.cfgに追加し、`en`を目的の言語コードに変更します。

```yaml
setr ox:locale en
```

localesディレクトリとあなたの言語用のファイルを作成します。

```json filename="locales/en.json"
{
  "grand_theft_auto": "grand theft auto",
  "male": "male",
  "female": "female",
  "suspect_sex": "suspect is %s"
}
```

```json filename="locales/fr.json"
{
  "grand_theft_auto": "vol de voiture",
  "male": "homme",
  "female": "femme",
  "suspect_sex": "le suspect est %s"
}
```

```lua filename="fxmanifest.lua"
files {
  'locales/*.json'
}
```

## 使用法

リソースでロケールモジュールを初期化します（1回）。

<Tabs items={["Lua", "JS"]}>
  <Tab>
  ```lua
  lib.locale()
  ```
  </Tab>
  <Tab>
  ```ts
  import {initLocale} from '@overextended/ox_lib/shared'

  initLocale()
  ```
  </Tab>
</Tabs>

新しいロケールグローバルで文字列をフォーマットします。
ロケール出力をフォーマットするために追加の引数を送信できます。

<Tabs items={['Lua', 'JS']}>
  <Tab>

    ```lua
    locale(str, ...)
    ```
  </Tab>
  <Tab>

    ```ts
    import { locale } from '@overextended/ox_lib/shared'

    locale(str, ...)
    ```
  </Tab>
</Tabs>

- str: `string`
- vararg?: `string` or `number`

<u>**Example**</u>

<Tabs items={['Lua', 'JS']}>
  <Tab>

  ```lua
  -- Load the locale module
  lib.locale()

  SetInterval(function()
      print(locale('grand_theft_auto'))
      print(locale('suspect_sex', locale('male')))
  end, 5000)
  ```
  </Tab>
  <Tab>

    ```ts
    import { initLocale, locale } from '@overextended/ox_lib/shared

    // Load the locale module
    initLocale()

    setInterval(() => {
      console.log(locale('grand_theft_auto'))
      console.log(locale('suspect_sex', locale('male')))
    }, 5000)
    ```
  </Tab>
</Tabs>


## フレーズ

ロケールを複数回呼び出すのではなく、他のロケールを参照するロケール文字列を作成してフレーズを構築できます。

```json
{
  "hello": "hello %s",
  "my_name_is": "my name is %s",
  "hello_my_name_is": "${hello}! ${my_name_is}."
}
```

<Tabs items={['Lua', 'JS']}>
  <Tab>

  ```lua
  print(locale('hello_my_name_is', 'doka', 'linden'))
  ```
  </Tab>
  <Tab>

    ```ts
    import { locale } from '@overextended/ox_lib/shared'

    console.log(locale('hello_my_name_is', 'doka', 'linden'))
    ```
  </Tab>
</Tabs>

## lib.getLocale

別のリソースからロケール文字列を取得し、辞書に追加します。

```lua
lib.getLocale(resource, key)
```

- resource: `string`
- key: `string`
