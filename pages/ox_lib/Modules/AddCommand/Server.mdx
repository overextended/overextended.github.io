# サーバー

コマンドを登録し、引数の検証、権限、チャットの提案を簡素化します。

```lua
lib.addCommand(commandName, properties, cb)
```

- commandName: `string` または `string[]`
- properties: `table` または `false`
  - help?: `string`
  - restricted?: `boolean` または `string` または `string[]`
  - params?: `table[]`
    - name: `string`
    - help?: `string`
    - type?: `'number'` または `'playerId'` または `'string'` または `'longString'`
    - optional?: `boolean`

```lua
lib.addCommand('giveitem', {
    help = 'プレイヤーにアイテムを与えます',
    params = {
        {
            name = 'target',
            type = 'playerId',
            help = 'ターゲットプレイヤーのサーバーID',
        },
        {
            name = 'item',
            type = 'string',
            help = '与えるアイテムの名前',
        },
        {
            name = 'count',
            type = 'number',
            help = '与えるアイテムの量、または1を与える場合は空白',
            optional = true,
        },
        {
            name = 'metatype',
            help = 'アイテムの "metadata.type" を設定します',
            optional = true,
        },
    },
    restricted = 'group.admin'
}, function(source, args, raw)
    local item = Items(args.item)

    if item then
        Inventory.AddItem(args.target, item.name, args.count or 1, args.metatype)
    end
end)
```
