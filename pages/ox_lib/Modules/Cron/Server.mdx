# サーバー

cronのLua実装。タスクを固定された時刻、日付、間隔で定期的に実行するようにスケジュールできます。

## Cron式

タスクを実行する時刻のセットを表す、空白で区切られた5つの値を含む文字列。

| フィールド        | 有効な値        |
| ------------ | ------------------- |
| 分      | `0-59`              |
| 時        | `0-23`              |
| 日にち | `1-31`              |
| 月        | `1-12` または `jan-dec` |
| 曜日  | `1-7` または `sun-sat`  |

**注意：** 曜日は `os.date` と一致するように設定されており、0から始まるcron標準とは異なり、1から始まります。

### `*` ワイルドカード
すべての値を表します。例：`* * * * *` は毎分実行され、`* * * * 1` は日曜日の毎分実行されます。

### `,` リスト
カンマを使用して値のリストを作成できます。例：`* * * * sun,mon,tue` は日曜日、月曜日、火曜日の毎分実行されます。

### `-` 範囲
ダッシュは値の範囲を定義します。例：`10-30 * * * *` は10分にタスクの実行を開始し、30分まで毎分実行されます。

### `/` ステップ
スラッシュはステップ値に使用できます。例：`* */4 * * *` は4時間ごとに実行され、`* 0,4,8,12,16,20 * * *` の省略形です。

## 関数

### lib.cron.new

新しい[cronジョブ](https://en.wikipedia.org/wiki/Cron)を作成し、タスクを固定された時刻または間隔で実行するようにスケジュールします。

```lua
lib.cron.new(expression, job, options)
```

- expression: `string`
  - 分、時、日、月、曜日を表す `* * * * *` などのcron式
- job: `fun(task: OxTask, date: osdate)`
- options?: `table`
  - debug?: `boolean`

戻り値:

- task: `OxTask`
