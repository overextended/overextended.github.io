# クライアント

メッセージの送信、URLの変更、削除をサポートする、シンプルで一元化されたDUI作成

## lib.dui:new

Duiのインスタンスを構築します。

```lua
lib.dui:new(data)
```

- data: `table`
  - url: `string`
  - width: `number`
  - height: `number`
  - debug?: `boolean`

戻り値:

- dui: `Dui`
  - url: `string`
  - duiObject: `number`
  - duiHandle: `string`
  - runtimeTxd: `number`
  - txdObject: `number`
  - dictName: `string`
  - txtName: `string`
  - setUrl: `function`(url: `string`)
  - sendMessage: `function`(data: `table`)
  - remove: `function`

### 使用例
```lua
local dui = lib.dui:new({
	url = ("nui://%s/web/index.html"):format(cache.resource), 
	width = 1920, 
	height = 1080,
	debug = true
})

-- URLの変更
dui:setUrl("https://google.com")

-- メッセージの送信
dui:sendMessage({
	action = "display",
	value = true
})

-- 破棄
dui:remove()
```
