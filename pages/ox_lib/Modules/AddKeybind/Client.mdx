# クライアント

キーバインドを登録し、キーバインドの相互作用を簡素化します。

## CKeybindクラス

次のプロパティを持つキーバインドを表すテーブル。

- name: `string`
- description: `string`
- currentKey: `string`
  - 現在のユーザーがこのキーバインドを設定しているキー
- disabled: `boolean`
  - キーバインドが現在無効になっているかどうか
- isPressed: `boolean`
  - キーバインドが押されているかどうかを示す状態
- hash: `number`
  - ゲーム自体の中で参照するために使用されるキーバインドの内部ハッシュ
- defaultKey?: `string`
  - 新規プレイヤー用にキーバインドを設定するデフォルトキー
  - 注：これを変更しても、既存のプレイヤーのキーは変更されません
- defaultMapper?: `string`
  - 詳細については、[入力マッパーID](https://docs.fivem.net/docs/game-references/input-mapper-parameter-ids/)を参照してください
- secondaryKey?: `string`
  - オプションのセカンダリキーバインド。
- secondaryMapper?: `string`
  - セカンダリキーのオプションのマッパー。それ以外の場合はデフォルトのマッパーを使用します。
- disable: `function(self: CKeybind, disable: boolean)`
  - キーバインドを有効/無効にする組み込み関数
- isControlPressed: `function(self: CKeybind)`
  - キーバインドの現在の押下状態を取得する組み込み関数
- onPressed?: `function(self: CKeybind)`
  - キーバインド押下時にトリガーされるユーザー定義関数
- onReleased?: `function(self: CKeybind)`
  - キーバインド解放時にトリガーされるユーザー定義関数

## lib.addKeybind

```lua
lib.addKeybind(data)
```

- data: `table`
  - name: `string`
  - description: `string`
  - defaultKey?: `string`
    - デフォルト：`None`
  - defaultMapper?: `string`
    - デフォルト：`keyboard`
  - secondaryKey?: `string`
  - secondaryMapper?: `string`
  - disabled?: `boolean`
    - キーバインドをデフォルトで無効にするかどうか
  - onPressed?: `function(self: CKeybind)`
    - キーバインド押下時にトリガーされる関数
  - onReleased?: `function(self: CKeybind)`
    - キーバインド解放時にトリガーされる関数

```lua
local keybind = lib.addKeybind({
    name = 'respects',
    description = 'Fキーを押して敬意を払います',
    defaultKey = 'F',
    onPressed = function(self)
        print(('pressed %s (%s)'):format(self.currentKey, self.name))
    end,
    onReleased = function(self)
        print(('released %s (%s)'):format(self.currentKey, self.name))
    end
})
```

### キーバインドの有効化/無効化

キーバインドは、disableメソッドを使用して有効化/無効化できます。

```lua
keybind:disable(true) -- キーバインドを無効にします
keybind:disable(false) -- キーバインドを有効にします
```

### キーバインドの押下状態

`lib.addKeybind()` で登録されたすべてのキーバインドは、押されているかどうかを示すブール状態を取得します。

次のメソッドで状態を取得できます。

```lua
local state = keybind:isControlPressed()
```
