# インターフェース

サーバーのテーマに合わせてUIのプライマリカラーを変更したい場合は、利用可能なconvarsを介して簡単に行うことができます。

UIをビルドする必要はなく、リソースを再起動するだけです。

Convars:

```yaml
setr ox:primaryColor blue
setr ox:primaryShade 8
```

プライマリカラーを変更すると、プログレスバー/サークル、スキルチェック、ラジアルメニューの中央ボタンとホバー、
ダイアログ確認ボタン、入力フィールドのフォーカスなどの要素の色が変更されます。

プリセットカラーとシェードの完全なリストは次の場所にあります。

https://v6.mantine.dev/theming/colors/#default-colors

独自のパレットを作成したい場合は、そのページのガイドに従うことをお勧めします。
これを行うには、ソースコードをダウンロードしてUIをビルドする_必要がある_ことに注意してください。
