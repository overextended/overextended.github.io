# 通知

## lib.notify

多くのスタイルオプションを備えたカスタム通知。

<Tabs items={["Lua", "JS"]}>
  <Tab>
    <Tabs items={["クライアント", "サーバー"]}>
      <Tab>
        ```lua
        lib.notify(data)
        ```
      </Tab>
      <Tab>
        ```lua
        TriggerClientEvent('ox_lib:notify', source, data)
        ```
      </Tab>
    </Tabs>
  </Tab>
  <Tab>
    ```ts
    import lib from '@overextended/ox_lib/client';

    lib.notify(data);
    ```
  </Tab>
</Tabs>

- id?: `string`
  - 設定すると、現在の通知は一意になり、スパムされたときに画面に1回だけ表示されます。
- title?: `string`
  - 説明がない場合は指定する必要があります
- description?: `string`
  - タイトルがない場合は指定する必要があります
  - Markdownサポート
- duration?: `number`
  - デフォルト: `3000`
- showDuration?: `boolean`
  - デフォルト: `true`
- position?: `'top'` または `'top-right'` または `'top-left'` または `'bottom'` または `'bottom-right'` または `'bottom-left'` または `'center-right'` または `'center-left'`
  - デフォルト: `'top-right'`
- type?: `'inform'` または `'error'` または `'success'`or `'warning'`
  - デフォルト: `'inform'`
- style?: `table` (`object`)
  - React CSS スタイル形式
- icon?: `string`
  - Font Awesome 6 アイコン名
- iconColor?: `string`
  - CSS Legal Color Values
- iconAnimation?: `'spin'` `'spinPulse'` `'spinReverse'` `'pulse'` `'beat'` `'fade'` `'beatFade'` `'bounce'` `'shake'`
- alignIcon?: `'top'` または `'center'`
  - デフォルト: `'center'`
- sound?: `table` (`object`)
  - bank?: `string`
    - 提供されたサウンドセットを含むオーディオバンクの名前
  - set: `string`
    - サウンド名が属するサウンドセット。
  - name: `string`
<Callout>
  `iconColor` を設定すると、対照的なアイコンの色とその円形の背景が削除されます。
</Callout>

## 使用例

### 標準

<Tabs items={["Lua", "JS"]}>
  <Tab>
    ```lua
    lib.notify({
        title = '通知タイトル',
        description = '通知の説明',
        type = 'success'
    })
    ```
  </Tab>
  <Tab>
    ```ts
    import lib from '@overextended/ox_lib/client';

    lib.notify({
      title: '通知タイトル',
      description: '通知の説明',
      type: 'success',
    });
    ```
  </Tab>
</Tabs>

### カスタム

<Tabs items={["Lua", "JS"]}>
  <Tab>
    ```lua
    lib.notify({
        id = 'some_identifier',
        title = '通知タイトル',
        description = '通知の説明',
        showDuration = false,
        position = 'top',
        style = {
            backgroundColor = '#141517',
            color = '#C1C2C5',
            ['.description'] = {
              color = '#909296'
            }
        },
        icon = 'ban',
        iconColor = '#C53030'
    })
    ```
  </Tab>
  <Tab>
    ```ts
    import lib from '@overextended/ox_lib/client';

    lib.notify({
      id: 'some_identifier',
      title: '通知タイトル',
      description: '通知の説明',
      showDuration: false,
      position: 'top',
      style: {
        backgroundColor: '#141517',
        color: '#C1C2C5',
        '.decription': {
          color: '#909296',
        },
      },
      icon: 'ban',
      iconColor: '#C53030',
    });
    ```
  </Tab>
</Tabs>
