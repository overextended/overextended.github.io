# クリップボード

## lib.setClipboard

プレイヤーのクリップボードを指定された文字列値に設定します。

<Callout>
    一部のNUIコンポーネントによってフォーカスがすでに取得されている場合は機能しません。
</Callout>

<Callout>
    新しい行を作成するには、文字列に `\n` だけでなく `\t\n` を使用します。
</Callout>

<Tabs items={["Lua", "JS"]}>
    <Tab>
        ```lua
        lib.setClipboard(value)
        ```
    </Tab>
    <Tab>
        ```ts
        import lib from '@overextended/ox_lib/client';

        lib.setClipboard(value);
        ```
    </Tab>
</Tabs>

- value: `string`
