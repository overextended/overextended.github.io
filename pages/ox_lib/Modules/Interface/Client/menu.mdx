# メニュー

特定のイベント関数を持つキーボードナビゲーションメニュー。

## lib.registerMenu

指定されたIDでメニューを登録およびキャッシュします。

<Tabs items={["Lua", "JS"]}>
  <Tab>
    ```lua
    lib.registerMenu(data, cb)
    ```
  </Tab>
  <Tab>
    ```ts
    import lib from '@overextended/ox_lib/client';

    lib.registerMenu(data, cb);
    ```
  </Tab>
</Tabs>

- data: `table` (`object`)
  - id: `string`
  - title: `string`
  - options: `table` (`array`)
    - label: `string`
    - progress?: `number`
    - colorScheme?: `string`
    - icon?: `string`
      - 左側に表示されるFontAwesomeアイコン。通知およびtextuiアイコンと同じように機能します。
      - 画像URL、pngおよびwebpファイルもサポートしていますが、FontAwesomeアイコンよりも使用することはお勧めしません。
    - iconColor?: `string`
    - iconAnimation?: `'spin'` `'spinPulse'` `'spinReverse'` `'pulse'` `'beat'` `'fade'` `'beatFade'` `'bounce'` `'shake'`
    - values?: `string[]` または `{ label: string, description: string }[]`
      - 提供されている場合、横スクロール可能なリストを作成します。
      - オブジェクトを使用し、説明を設定すると、設定された説明がメニューのツールチップに表示されます。
    - checked?: `boolean`
      - trueまたはfalseのいずれかに設定すると、ボタンがチェックボックスになります。`values`も提供されている場合は、ボタンは
        スクロール可能なリストになります。
    - description?: `string`
      - 提供された説明とともに、ホバーされたアイテムのメニューの下にツールチップを表示します。
    - defaultIndex?: `number`
      - リストの現在のインデックスを指定された数値に設定します。
    - args?: `{[string]: any}`
      - ボタンを介して任意の引数を渡すことができます。
      - ボタンに `values` がある場合、`isScroll` が自動的に渡されます。
      - ボタンに `checked` がtrueまたはfalseのいずれかに設定されている場合、`isCheck` が自動的に渡されます。
    - close?: `boolean`
      - falseに設定すると、このオプションと対話してもメニューは閉じません。
  - position?: `'top-left'` または `'top-right'` または `'bottom-left'` または `'bottom-right'`
    - デフォルト: `'top-left'`
  - disableInput?: `boolean`
    - デフォルト: `false`
  - canClose: `boolean`
    - falseに設定すると、ユーザーはいずれかのボタンを押さずにメニューを終了できなくなります。
  - onClose: `function`(keyPressed?: `'Escape' | 'Backspace'`)
    - ESC/Backspaceを介してメニューが終了したときに実行される関数。
  - onSelected: `function`(selected: `number`, secondary: `number` | `boolean`, args: `{[string]: any}`)
    - メニューで選択されたボタンが変更されたときに実行される関数。
  - onSideScroll: `function`(selected: `number`, scrollIndex: `number`, args: `{[string]: any}`)
    - スクロールリストアイテムが変更されるたびに実行される関数。
  - onCheck: `function`(selected: `number`, checked: `boolean`, args: `{[string]: any}`)
    - チェックボックスが切り替えられるたびに実行される関数。
- cb: `function`(selected: `number`, scrollIndex: `number`, args: `{[string]: any}`)
  - メニュー項目が押されたときのコールバック関数。

## lib.showMenu

提供されたIDでメニューを表示します。

<Tabs items={["Lua", "JS"]}>
  <Tab>
    ```lua
    lib.showMenu(id)
    ```
  </Tab>
  <Tab>
    ```ts
    import lib from '@overextended/ox_lib/client';

    lib.showMenu(id);
    ```
  </Tab>
</Tabs>

- id: `string`

## lib.hideMenu

<Tabs items={["Lua", "JS"]}>
  <Tab>
    ```lua
    lib.hideMenu(onExit)
    ```
  </Tab>
  <Tab>
    ```ts
    import lib from '@overextended/ox_lib/client';

    lib.hideMenu(onExit);
    ```
  </Tab>
</Tabs>

- onExit?: `boolean`
  - trueの場合、メニューの `onClose` 関数を実行します。

## lib.setMenuOptions

<Tabs items={["Lua", "JS"]}>
  <Tab>
    ```lua
    lib.setMenuOptions(id, options, index)
    ```
  </Tab>
  <Tab>
    ```ts
    import lib from '@overextended/ox_lib/client';

    lib.setMenuOptions(id, options, index);
    ```
  </Tab>
</Tabs>

- id: `string`
- options: `table` (`object` or `array`)
- index?: `number`
  - 指定されている場合、指定されたオプションインデックスのオプションテーブルのみを設定します。

<u>**例:**</u><br />
指定されたメニューの3番目のインデックスオプションを置き換えます

<Tabs items={["Lua", "JS"]}>
  <Tab>
    ```lua
    lib.setMenuOptions('some_menu_id', {label = 'New option', icon = 'plus'}, 3)
    ```
  </Tab>
  <Tab>
    ```ts
    import lib from '@overextended/ox_lib/client';

    lib.setMenuOptions('some_menu_id', { label: 'New option', icon: 'plus' }, 3);
    ```
  </Tab>
</Tabs>

## 使用例

まず、指定されたオプションでメニューを登録し、次にコマンドでshow関数を呼び出します。

<Callout>
  外部の値に依存しないメニュー（別名静的メニュー）を常に再登録しないでください。
</Callout>

<Tabs items={["Lua", "JS"]}>
  <Tab>
    ```lua
    lib.registerMenu({
        id = 'some_menu_id',
        title = 'メニュータイトル',
        position = 'top-right',
        onSideScroll = function(selected, scrollIndex, args)
            print("スクロール: ", selected, scrollIndex, args)
        end,
        onSelected = function(selected, secondary, args)
            if not secondary then
                print("通常のボタン")
            else
                if args.isCheck then
                    print("チェックボタン")
                end

                if args.isScroll then
                    print("スクロールボタン")
                end
            end
            print(selected, secondary, json.encode(args, {indent=true}))
        end,
        onCheck = function(selected, checked, args)
            print("チェック: ", selected, checked, args)
        end,
        onClose = function(keyPressed)
            print('メニューが閉じられました')
            if keyPressed then
                print(('%s を押してメニューを閉じました'):format(keyPressed))
            end
        end,
        options = {
            {label = 'シンプルなボタン', description = '説明があります！'},
            {label = 'チェックボックスボタン', checked = true},
            {label = 'アイコン付きスクロールボタン', icon = 'arrows-up-down-left-right', values={'こんにちは', 'そこ'}},
            {label = '引数付きボタン', args = {someArg = 'nice_button'}},
            {label = 'リストボタン', values = {'あなたは', 'できます', '横に', 'スクロール', 'これ'}, description = '説明もあります！'},
            {label = 'デフォルトインデックス付きリストボタン', values = {'あなたは', 'できます', '横に', 'スクロール', 'これ'}, defaultIndex = 5},
            {label = '引数付きリストボタン', values = {'あなたは', 'できます', '横に', 'スクロール', 'これ'}, args = {someValue = 3, otherValue = 'value'}},
        }
    }, function(selected, scrollIndex, args)
        print(selected, scrollIndex, args)
    end)

    RegisterCommand('testmenu', function()
        lib.showMenu('some_menu_id')
    end)
    ```
  </Tab>
  <Tab>
    ```ts
    import lib from '@overextended/ox_lib/client';

    lib.registerMenu(
      {
        id: 'some_menu_id',
        title: 'メニュータイトル',
        position: 'top-right',
        onSideScroll: (selected, scrollIndex, args) => {
          console.log('スクロール: ', selected, scrollIndex, args);
        },
        onSelected: (selected, secondary, args) => {
          if (!secondary) {
            console.log('通常のボタン');
          } else {
            if (args.isCheck) {
              console.log('チェックボタン');
            }

            if (args.isScroll) {
              console.log('スクロールボタン');
            }
          }
          console.log(selected, secondary, JSON.stringify(args, null, 2));
        },
        onCheck: (selected, checked, args) => {
          console.log('チェック: ', selected, checked, args);
        },
        onClose: (keyPressed) => {
          console.log('メニューが閉じられました');
          if (keyPressed) {
            console.log(`${keyPressed} を押してメニューを閉じました`);
          }
        },
        options: [
          { label: 'シンプルなボタン', description: '説明があります！' },
          { label: 'チェックボックスボタン', checked: true },
          { label: 'アイコン付きスクロールボタン', icon: 'arrows-up-down-left-right', values: ['こんにちは', 'そこ'] },
          { label: '引数付きボタン', args: { someArg: 'nice_button' } },
          {
            label: 'リストボタン',
            values: ['あなたは', 'できます', '横に', 'スクロール', 'これ'],
            description: '説明もあります！',
          },
          { label: 'デフォルトインデックス付きリストボタン', values: ['あなたは', 'できます', '横に', 'スクロール', 'これ'], defaultIndex: 5 },
          {
            label: '引数付きリストボタン',
            values: ['あなたは', 'できます', '横に', 'スクロール', 'これ'],
            args: { someValue: 3, otherValue: 'value' },
          },
        ],
      },
      (selected, scrollIndex, args) => {
        console.log(selected, scrollIndex, args);
      }
    );

    RegisterCommand(
      'testmenu',
      () => {
        lib.showMenu('some_menu_id');
      },
      false
    );
    ```
  </Tab>
</Tabs>
