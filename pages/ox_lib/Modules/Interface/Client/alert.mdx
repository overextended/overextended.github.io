# アラートダイアログ

プレイヤーにメッセージを表示できるシンプルなアラートダイアログ。
プレイヤーが確認ボタンを押したか、ダイアログをキャンセルしたかを返します。

## lib.alertDialog

<Tabs items={["Lua", "JS"]}>
  <Tab>
    <Tabs items={["クライアント", "サーバー"]}>
      <Tab>
        ```lua
        lib.alertDialog(data)
        ```
      </Tab>
      <Tab>
        ```lua
        TriggerClientEvent('ox_lib:alertDialog', source, data)
        ```
      </Tab>
    </Tabs>
  </Tab>
  <Tab>
    <Callout>
      この関数は**非同期**であり、プロミスに対して `.then` コールバックを行うか、関数を `async` にする必要があります。
    </Callout>

    ```ts
    import lib from '@overextended/ox_lib/client';

    lib.alertDialog(data);
    ```
  </Tab>
</Tabs>

- data: `table` (`object`)
  - header: `string`
    - ダイアログのタイトル。
  - content: `string`
    - ダイアログの本文コンテンツ。マークダウンをサポートします。
  - centered?: `boolean`
    - ダイアログを垂直方向および水平方向に中央揃えします。
  - cancel?: `boolean`
    - キャンセルボタンを表示します（これが定義されていない場合でもESCは使用可能です）。
  - size?: `'xs'` または `'sm'` または `'md'` または `'lg'` または `'xl'`
  - overflow?: `boolean`
  - labels?: `table`
    - キャンセルボタンおよび/または確認ボタンの表示ラベルを定義できます。
    - cancel?: `string`
    - confirm?: `string`

プレイヤーが確認ボタンを押した場合は `'confirm'` を返します。それ以外の場合、プレイヤーがキャンセルボタンを押したか、ESCでダイアログを終了した場合は `'cancel'` を返します。

## lib.closeAlertDialog

アクティブなアラートダイアログを強制的に閉じ、その戻り値を `nil` に設定します

<Tabs items={["Lua", "JS"]}>
  <Tab>
    ```lua
    lib.closeAlertDialog()
    ```
  </Tab>
  <Tab>
    ```ts
    import lib from '@overextended/ox_lib/client';

    lib.closeAlertDialog();
    ```
  </Tab>
</Tabs>

## 例

<Tabs items={["Lua", "JS"]}>
  <Tab>
    ```lua
    local alert = lib.alertDialog({
        header = 'こんにちは',
        content = 'ケノービ将軍  \n マークダウン対応！',
        centered = true,
        cancel = true
    })

    print(alert)
    ```
  </Tab>
  <Tab>
    ```ts
    import lib from '@overextended/ox_lib/client';

    const alert = await lib.alertDialog({
      header: 'こんにちは',
      content: 'ケノービ将軍  \n マークダウン対応！',
      centered: true,
      cancel: true,
    });

    console.log(alert);
    ```
  </Tab>
</Tabs>
