# プログレス

## lib.progressBar

実行中のプログレスバーを表示します。

<Tabs items={["Lua", "JS"]}>
  <Tab>
    ```lua
    lib.progressBar(data)
    ```
  </Tab>
  <Tab>
    ```ts
    import lib from '@overextended/ox_lib/client'

    lib.progressBar(data)
    ```
  </Tab>
</Tabs>

- duration: `number`
- label: `string`
- useWhileDead?: `boolean`
- allowRagdoll?: `boolean`
- allowSwimming?: `boolean`
- allowCuffed?: `boolean`
- allowFalling?: `boolean`
- canCancel?: `boolean`
- anim?: `table` (`object`)
  - dict?: `string`
    - シナリオまたはdictのいずれかを指定する必要があります
  - clip: `string`
  - flag?: `number`
    - デフォルト: `49`
  - blendIn?: `float`
    - デフォルト: `3.0`
  - blendOut?: `float`
    - デフォルト: `1.0`
  - duration?: `number`
    - デフォルト: `-1`
  - playbackRate?: `number`
    - デフォルト: `0`
  - lockX?: `boolean`
  - lockY?: `boolean`
  - lockZ?: `boolean`
  - scenario?: `string`
    - シナリオまたはdictのいずれかを指定する必要があります
  - playEnter?: `boolean`
    - デフォルト: `true`
- prop?: `table` (`object` または `array`)
  - [ 複数のプロップを定義する場合は、個別のテーブル（オブジェクトの配列）として渡すことができます ]
  - model: `hash`
  - bone?: `number`
    - デフォルト: `60309`
  - pos: `table`
    - x: `number`
    - y: `number`
    - z: `number`
  - rot: `table` (`object`)
    - x: `number`
    - y: `number`
    - z: `number`
  - rotOrder?: `number`
    - [ヨー、ピッチ、ロールが適用される順序。](https://docs.fivem.net/natives/?_0xAFBD61CC738D9EB9)
    - デフォルト: `0`
- disable?: `table` (`object`)
  - move?: `boolean`
  - car?: `boolean`
  - combat?: `boolean`
  - mouse?: `boolean`
  - sprint?: `boolean`

### 使用例

<Tabs items={["Lua", "JS"]}>
  <Tab>
    ```lua
    if lib.progressBar({
        duration = 2000,
        label = '水を飲んでいます',
        useWhileDead = false,
        canCancel = true,
        disable = {
            car = true,
        },
        anim = {
            dict = 'mp_player_intdrink',
            clip = 'loop_bottle'
        },
        prop = {
            model = `prop_ld_flow_bottle`,
            pos = vec3(0.03, 0.03, 0.02),
            rot = vec3(0.0, 0.0, -1.5)
        },
    }) then print('完了時に何かを実行') else print('キャンセル時に何かを実行') end
    ```
  </Tab>
  <Tab>
    <Callout>
      この関数は**非同期**であり、プロミスに対して `.then` コールバックを行うか、関数を `async` にする必要があります。
    </Callout>

    ```ts
    import lib from '@overextended/ox_lib/client'

    if (await lib.progressBar({
      duration: 2000,
      label = '水を飲んでいます',
      useWhileDead: false,
      canCancel: true,
      disable: {
        car: true,
      },
      anim: {
        dict: 'mp_player_intdrink',
        clip: 'loop_bottle'
      },
      prop: {
        model: `prop_ld_flow_bottle`,
        pos: {x: 0.03, y: 0.03, z: 0.02},
        rot: {x: 0.0, y: 0.0, z: -1.5}
      },
    })) console.log('完了時に何かを実行');
    else console.log('キャンセル時に何かを実行')
    ```
  </Tab>
</Tabs>

## lib.progressActive

プログレスバーが現在アクティブな場合はtrueを返します。

<Tabs items={["Lua", "JS"]}>
  <Tab>
    ```lua
    lib.progressActive()
    ```
  </Tab>
  <Tab>
    ```ts
    import lib from '@overextended/ox_lib/client'

    lib.progressActive()
    ```
  </Tab>
</Tabs>

## lib.cancelProgress

プログレスバーがアクティブで、
プログレスバーをキャンセルできる場合はキャンセルします。

<Tabs items={["Lua", "JS"]}>
  <Tab>
    ```lua
    lib.cancelProgress()
    ```
  </Tab>
  <Tab>
    ```ts
    import lib from '@overextended/ox_lib/client'

    lib.cancelProgress()
    ```
  </Tab>
</Tabs>
