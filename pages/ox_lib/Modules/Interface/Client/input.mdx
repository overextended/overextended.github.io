# 入力ダイアログ

入力ダイアログウィンドウでは、入力フィールドを設定することでユーザーからデータを取得できます。

## lib.inputDialog

<Tabs items={["Lua", "JS"]}>
  <Tab>
    ```lua
    lib.inputDialog(heading, rows, options)
    ```
  </Tab>
  <Tab>
    ```ts
    import lib from '@overextended/ox_lib/client';

    lib.inputDialog(heading, rows, options);
    ```
  </Tab>
</Tabs>

- heading: `string`
- rows: `string[]` または `table` (`array`)
  - type: `'input'` または `'number'` または `'checkbox'` または `'select'` または `'slider'` または `'color'` または `'multi-select'` または `'date'` または `'date-range'` または `'time'` または `'textarea'`
- options?: `table`(`object`)
  - allowCancel: `boolean`
    - falseの場合、ユーザーは送信されるまで入力ダイアログをキャンセルして閉じることができません。
    - 定義されていない場合、ユーザーは入力ダイアログをキャンセルして閉じることができます。

### フィールドタイプのプロパティ

- input

  - label: `string`
  - description?: `string`
  - placeholder?: `string`
  - icon?: `string`
  - required? `boolean`
  - disabled?: `boolean`
  - default?: `string`
  - password?: `boolean`
  - min?: `number`
  - max?: `number`

- number

  - label: `string`
  - description?: `string`
  - placeholder?: `string`
  - icon?: `string`
  - required? `boolean`
  - disabled?: `boolean`
  - default?: `number`
  - min?: `number`
  - max?: `number`
  - precision?:  `number`
  - step?: `number`

- checkbox

  - label: `string`
  - checked?: `boolean`
  - disabled?: `boolean`
  - required?: `boolean`

- select および multi-select

  - label: `string`
  - options: `table`(`array`)
    - value: `string`
    - label?: `string`
  - description?: `string`
  - placeholder?: `string`
  - icon?: `string`
  - required? `boolean`
  - disabled?: `boolean`
  - default?: `string` | `table` (`multi-select` のみ)
    - デフォルトオプションの値。
  - clearable?: `boolean`
  - searchable?: `boolean`
  - maxSelectedValues?: `number` (`multi-select` のみ)
    - 選択できるオプションの最大数。

- slider

  - label: `string`
  - placeholder?: `string`
  - icon?: `string`
  - required? `boolean`
  - disabled?: `boolean`
  - default?: `number`
  - min?: `number`
  - max?: `number`
  - step?: `number`

- color

  - label: `string`
  - description?: `string`
  - placeholder?: `string`
  - icon?: `string`
  - required? `boolean`
  - disabled?: `boolean`
  - default?: `string`
  - format?: `'hex'` | `'hexa'` | `'rgb'` | `'rgba'` | `'hsl'` | `'hsla'`;

- date

  - label: `string`
  - description?: `string`
  - icon?: `string`
  - required? `boolean`
  - disabled?: `boolean`
  - default?: `string` または `true`
    - Trueは現在の日付をデフォルトとします
  - format?: `string`
    - フィールドに表示する日付形式
  - returnString?: `boolean`
    - 日付を文字列として返します。デフォルト形式は `DD/MM/YYYY` ですが、`format` が定義されている場合はそれを使用します。
  - clearable?: `boolean`
  - min?: `string`
    - "01/01/2000"
  - max?: `string`
    - "12/12/2023"

- date-range

  - label: `string`
  - description?: `string`
  - icon?: `string`
  - required? `boolean`
  - disabled?: `boolean`
  - default?: `[string, string]`
  - format?: `string`
    - フィールドに表示する日付形式
  - returnString?: `boolean`
    - 日付を文字列として返します。デフォルト形式は `DD/MM/YYYY` ですが、`format` が定義されている場合はそれを使用します。
  - clearable?: `boolean`

- time

  - label: `string`
  - description?: `string`
  - icon?: `string`
  - required? `boolean`
  - disabled?: `boolean`
  - default?: `string`
  - format?: `'12'` または `'24'`
  - clearable?: `boolean`

- textarea
  - label: `string`
  - description?: `string`
  - placeholder?: `string`
  - icon?: `string`
  - required? `boolean`
  - disabled?: `boolean`
  - default?: `number`
  - min?: `number`
    - テキストエリアが取る最小行数。
  - max?: `number`
    - テキストエリアが取る最大行数。超えるとオーバーフローします。
  - autosize?: `boolean`
    - trueの場合、最大行数に達するまでコンテンツとともにテキストエリアが拡大します。

コールバックデータはプロミスベースであり、ユーザーがデータを送信するかポップアップを終了するまでスレッドの実行は継続されません。

返されるデータはテーブル（配列）であり、インデックスはダイアログに送信された行を表します。したがって、最初のフィールドからデータを取得したい場合はインデックス `1`（`0`）、3番目のフィールドからデータを取得したい場合はインデックス `3`（`2`）などになります...

<Callout>
  `date`、`date-range`、`time` などのフィールドタイプは、設定値のUNIXタイムスタンプを返します。
</Callout>

## lib.closeInputDialog

アクティブな入力ダイアログを強制的に閉じ、その戻り値を `nil` に設定します。

<Tabs items={["Lua", "JS"]}>
  <Tab>
    ```lua
    lib.closeInputDialog()
    ```
  </Tab>
  <Tab>
    ```ts
    import lib from '@overextended/ox_lib/client';

    lib.closeInputDialog();
    ```
  </Tab>
</Tabs>

## Usage Example

### Basic

<Tabs items={["Lua", "JS"]}>
  <Tab>
    ```lua
    local input = lib.inputDialog('基本ダイアログ', {'最初の行', '2番目の行'})

    if not input then return end
    print(json.encode(input), input[1], input[2])
    ```
  </Tab>
  <Tab>
    <Callout>
      この関数は**非同期**であり、プロミスに対して `.then` コールバックを行うか、関数を `async` にする必要があります。
    </Callout>

    ```ts
    const input = await lib.inputDialog('基本ダイアログ', ['最初の行', '2番目の行']);

    if (!input) return;
    console.log(input, input[0], input[1]);
    ```
  </Tab>
</Tabs>

### 詳細設定

<Tabs items={["Lua", "JS"]}>
  <Tab>
    ```lua
    local input = lib.inputDialog('ダイアログのタイトル', {
      {type = 'input', label = 'テキスト入力', description = 'いくつかの入力説明', required = true, min = 4, max = 16},
      {type = 'number', label = '数値入力', description = 'いくつかの数値説明', icon = 'hashtag'},
      {type = 'checkbox', label = 'シンプルなチェックボックス'},
      {type = 'color', label = 'カラー入力', default = '#eb4034'},
      {type = 'date', label = '日付入力', icon = {'far', 'calendar'}, default = true, format = "DD/MM/YYYY"}
    })

    print(json.encode(input))

    -- カラーピッカーからRGB値を取得
    local rgb = lib.math.torgba(input[4])

    -- Luaのosライブラリ（サーバーのみ）を使用して日付タイムスタンプを読み取り可能な形式に変換
    local timestamp = math.floor(input[5] / 1000)
    local date = os.date('%Y-%m-%d %H:%M:%S', timestamp)
    ```
  </Tab>
  <Tab>
    <Callout>
      この関数は**非同期**であり、プロミスに対して `.then` コールバックを行うか、関数を `async` にする必要があります。
    </Callout>

    ```ts
    const input = await lib.inputDialog('警察ロッカー', [
      { type: 'input', label: 'テキスト入力', description: 'いくつかの入力説明', required = true, min = 3, max = 16 },
      { type: 'number', label: '数値入力', description: 'いくつかの数値説明', icon = 'hashtag' },
      { type: 'checkbox', label: 'シンプルなチェックボックス' },
      { type: 'color', label: 'カラー入力', default = '#eb4034' },
      { type: 'date', label: '日付入力', icon: ['far', 'calendar'], default = true, format: 'DD/MM/YYYY' },
    ]);

    console.log(JSON.stringify(input, null, 2));

    -- カラーピッカーからr、g、bの値を取得
    const regExp = /rgb\((\d+),\s*(\d+),\s*(\d+)\)/;
    const colourInput = input[3];

    const color = regExp.exec(colourInput);

    if (!color) return;

    console.log(+color[1], +color[2], +color[3]);
    ```
  </Tab>
</Tabs>
