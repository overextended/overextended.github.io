# ラジアルメニュー

- ラジアルメニューにはグローバルメニューがあり、デフォルトでは `z` でアクセスでき、少なくとも1つのアイテムがある場合にのみ表示されます。
- `lib.addRadialItem` および `lib.removeRadialItem` を使用して、グローバルメニューからアイテムを追加および削除できます。
- サブメニューを作成するには `lib.registerRadial` を使用し、アイテムの `menu` プロパティを使用してそれらのサブメニューを開きます。

## lib.addRadialItem

グローバルラジアルメニューに追加されたアイテムまたはアイテムの配列。

<Tabs items={["Lua", "JS"]}>
  <Tab>
    ```lua
    lib.addRadialItem(items)
    ```
  </Tab>
  <Tab>
    ```ts
    import lib from '@overextended/ox_lib/client';

    lib.addRadialItem(items);
    ```
  </Tab>
</Tabs>

- items: `table` (`object` または `array`)
  - id: `string`
    - オプションを削除するために使用されるID。
  - icon: `string`
    - Font AwesomeまたはカスタムURIのいずれか。1
  - iconWidth?: `number`
  - iconHeight?: `number`
    - カスタムURIの場合、アイコンのサイズを調整します。
  - label: `string`
    - ラベルは `  \n` を使用して改行を挿入します
  - menu?: `string`
    - 開くメニューのID。
  - onSelect: `function(currentMenu: string | nil, itemIndex: number)` | `string`
    - ユーザーがアイテムをクリックしたときに実行される関数。
  - keepOpen?: `boolean`

## lib.registerRadial

事前定義されたオプションを持つラジアルサブメニューを登録します。

<Tabs items={["Lua", "JS"]}>
  <Tab>
    ```lua
    lib.registerRadial(radial)
    ```
  </Tab>
  <Tab>
    ```ts
    import lib from '@overextended/ox_lib/client';

    lib.registerRadial(radial);
    ```
  </Tab>
</Tabs>

- radial: `table` (`object`)
  - id: `string`
    - アイテムの `menu` プロパティで開くために使用される一意のメニューID。
  - items: `array`
    - icon: `string`
    - label: `string`
      - ラベルは `  \n` を使用して改行を挿入します
    - menu?: `string`
      - 開くメニューのID。
    - onSelect?: `function(currentMenu: string | nil, itemIndex: number)` | `string`
      - ユーザーがアイテムをクリックしたときに実行される関数。

<Tabs items={["Lua", "JS"]}>

## 使用例

<Callout>
  グローバルメニューかサブメニューかにかかわらず、ラジアルメニュー項目を追加する場合は、できるだけ短いラベルを使用するようにしてください。
  長いラベルは見栄えが悪く、密度が高いためラジアルメニューでは使用しないでください。
</Callout>

libのポイントシステムを利用したグローバルオプションとオプションを含むユースケースの例を次に示します。

<Tabs items={["Lua", "JS"]}>
  <Tab>
    ```lua
    exports('myMenuHandler', function(menu, item)
        print(menu, item)

        if menu == 'police_menu' and item == 1 then
            print('手錠')
        end
    end)

    lib.registerRadial({
      id = 'police_menu',
      items = {
        {
          label = '手錠',
          icon = 'handcuffs',
          onSelect = 'myMenuHandler'
        },
        {
          label = '身体検査',
          icon = 'hand'
        },
        {
          label = '指紋',
          icon = 'fingerprint'
        },
        {
          label = '拘留',
          icon = 'bus'
        },
        {
          label = '捜索',
          icon = 'magnifying-glass',
          onSelect = function()
            print('捜索')
          end
        }
      }
    })

    lib.addRadialItem({
      {
        id = 'police',
        label = '警察',
        icon = 'shield-halved',
        menu = 'police_menu'
      },
      {
        id = 'business_stuff',
        label = 'ビジネス',
        icon = 'briefcase',
        onSelect = function()
          print("ビジネス")
        end
      }
    })

    local coords = GetEntityCoords(cache.ped)
    local point = lib.points.new(coords, 5)

    function point:onEnter()
      lib.addRadialItem({
        id = 'garage_access',
        icon = 'warehouse',
        label = 'ガレージ',
        onSelect = function()
          print('ガレージ')
        end
      })
    end

    function point:onExit()
      lib.removeRadialItem('garage_access')
    end
    ```
  </Tab>
