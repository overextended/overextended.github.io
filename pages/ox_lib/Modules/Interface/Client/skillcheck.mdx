# スキルチェック

## lib.skillCheck

定義された難易度でスキルチェックを実行します。

<Tabs items={["Lua", "JS"]}>
  <Tab>
    ```lua
    lib.skillCheck(difficulty, inputs)
    ```
  </Tab>
  <Tab>
    ```ts
    import lib from '@overextended/ox_lib/client';

    lib.skillCheck(difficulty, inputs);
    ```

  </Tab>
</Tabs>

- difficulty: `'easy'` または `'medium'` または `'hard'` または `table`
  - プリセット難易度:
    - 'easy' - `{ areaSize: 50, speedMultiplier: 1 }`
    - 'medium' - `{ areaSize: 40, speedMultiplier: 1.5 }`
    - 'hard' - `{ areaSize: 25, speedMultiplier: 1.75 }`
  - 上記のプリセット文字列の代わりにオブジェクトを送信することで、カスタム難易度を設定できます
    - areaSize: `number`
      - 成功エリアのサイズ（度単位）
    - speedMultiplier: `number`
      - インジケーターの速度の乗数
- inputs?: `string[]`
  - 各スキルチェックに対して、入力テーブルからランダムなキーが選択されます
  - 入力が定義されていない場合、キーはデフォルトで `e` になります

## lib.skillCheckActive

スキルチェックが現在アクティブな場合はtrueを返します。

<Tabs items={["Lua", "JS"]}>
  <Tab>
    ```lua
    lib.skillCheckActive()
    ```
  </Tab>
  <Tab>
    ```ts
    import lib from '@overextended/ox_lib/client'

    lib.skillCheckActive()
    ```

  </Tab>
</Tabs>

## lib.cancelSkillCheck

現在進行中のスキルチェックをキャンセルします。

<Tabs items={["Lua", "JS"]}>

  </Tab>
  <Tab>
    ```ts

    ```

  </Tab>
</Tabs>

## 使用例

<Tabs items={["Lua", "JS"]}>
  <Tab>
    ```lua
    local success = lib.skillCheck({'easy', 'easy', {areaSize = 60, speedMultiplier = 2}, 'hard'}, {'w', 'a', 's', 'd'})
    ```
  </Tab>
  <Tab>
    ```ts
    import lib from '@overextended/ox_lib/client';

    const success = await lib.skillCheck(
      ['easy', 'easy', { areaSize: 60, speedMultiplier: 2 }, 'hard'],
      ['w', 'a', 's', 'd']
    );
    ```

  </Tab>
</Tabs>
