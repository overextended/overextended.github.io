# テキストUI

## lib.showTextUI

テキストUIウィンドウを表示します。

<Callout type="warning">
  この関数を毎ティック実行しないでください。トグルとして使用することを目的としています。
</Callout>

<Tabs items={["Lua", "JS"]}>
  <Tab>
    ```lua
    lib.showTextUI(text, options)
    ```
  </Tab>
  <Tab>
    ```ts
    import lib from '@overextended/ox_lib/client';

    lib.showTextUI(text, options);
    ```
  </Tab>
</Tabs>

- text: `string`
- options?: `table`
  - position?: `'right-center'` または `'left-center'` または `'top-center'` または `'bottom-center'`
    - デフォルト: `'right-center'`
  - icon?: `string` または `table` (`array`)
  - iconColor?: `string`
  - iconAnimation?: `'spin'` `'spinPulse'` `'spinReverse'` `'pulse'` `'beat'` `'fade'` `'beatFade'` `'bounce'` `'shake'`
  - style?: React.CSSProperties
  - alignIcon?: `'top'` または `'center'`
    - デフォルト: `'center'`

## lib.hideTextUI

現在表示されているテキストUIウィンドウを非表示にします

<Tabs items={["Lua", "JS"]}>
  <Tab>
    ```lua
    lib.hideTextUI()
    ```
  </Tab>
  <Tab>
    ```ts
    import lib from '@overextended/ox_lib/client';

    lib.hideTextUI();
    ```
  </Tab>
</Tabs>

## lib.isTextUIOpen

テキストUIが開いているかどうかを返します。現在表示されているテキストは2番目の値として返されます。

<Tabs items={["Lua", "JS"]}>
  <Tab>
    ```lua
    local isOpen, text = lib.isTextUIOpen()
    ```
  </Tab>
  <Tab>
    ```ts
    import lib from '@overextended/ox_lib/client';

    const [isOpen, text] = lib.isTextUIOpen();
    ```
  </Tab>
</Tabs>

### カスタムスタイリング

  <Tab>

    ```lua
        position = "top-center",

        style = {

            color = 'white'
        }
    })
    ```
  </Tab>

    ```ts
      },
    ```

  </Tab>
