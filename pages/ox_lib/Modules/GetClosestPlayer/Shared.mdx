# 共有

## lib.getClosestPlayer

座標のセットに最も近いプレイヤーのプレイヤーID、歩行者ハンドル、および座標を取得します。

```lua
lib.getClosestPlayer(coords, maxDistance, includePlayer)
```

- coords: `vector3`
  - チェックする座標。
- maxDistance?: `number`
  - チェックする最大距離。
  - デフォルト: `2.0`
- includePlayer?: `boolean`
  - 現在のプレイヤーを含めるかどうか。サーバーでは無視されます。
  - デフォルト: `false`

戻り値:

- playerId?: `number`
- playerPed?: `number`
- playerCoords?: `vector3`
