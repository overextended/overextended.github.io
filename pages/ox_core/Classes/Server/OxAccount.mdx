## OxAccount

- accountId` number`


## OxAccountMetadata

- id: `number`
- balance: `number`
- isDefault: `number`
- label?: `boolean`
- owner?: `number`
- group?: `string`
- type: `personal` | `shared` | `group`

## OxAccount.get

アカウントのメタデータから特定のキーの値を取得します。

```lua
account.get(key)
```

**パラメータ**

- key: `string` | `string[]`

**戻り値**

- `OxAccountMetadata[key]` | `OxAccountMetadata[key][]`
  - 複数のキーを渡すと配列を返します。


## OxAccount.addBalance

アカウントに資金を追加します。

```lua
account.addBalance(data)
```

**パラメータ**

- data: `object`
  - amount: `number`
  - message?: `string`

**戻り値**

- `object`
  - success: `boolean`
  - message: `'amount_not_number'` | `'no_balance'` | `'something_went_wrong'`

<Tabs items={['Lua']}>
  <Tab>
    ```lua
    account.addBalance({ amount = 1000, message = 'Paycheck' })
    ```

  </Tab>
</Tabs>


## OxAccount.removeBalance

アカウントから資金を引き出します。

```lua
account.removeBalance(data)
```

**パラメータ**

- `object`
  - amount: `number`
  - message?: `string`
  - overdraw?: `boolean`
    - 十分な残高がない場合でも残高を削除します（マイナス残高）

**戻り値**

- `object`
  - success: `boolean`
  - message: `'amount_not_number'` | `'no_balance'` | `'something_went_wrong'`

<Tabs items={['Lua']}>
  <Tab>
    ```lua
    account.removeBalance({ amount = 1000, message = 'Impound', overdraw = true })
    ```

  </Tab>
</Tabs>

## OxAccount.transferBalance

別のアカウントに資金を送金します。

```lua
account.transferBalance(data)
```

**パラメータ**

- `object`
  - toId: `number`
    - 送金先のアカウントID。
  - amount: `number`
  - message?: `string`
    - 受信者へのメッセージ
  - overdraw?: `boolean`
  - note?: `string`
    - 送金者へのメモ
  - actorId? `number`
    - 送金を開始したプレイヤーのcharId。

**戻り値**

- `object`
  - success: `boolean`
  - message: `'amount_not_number'` | `'no_balance'` | `'something_went_wrong'`

<Tabs items={['Lua']}>
  <Tab>
    ```lua
    account.transferBalance({
      toId = 100000000,
      amount = 20,
      message = 'Impound charge for plate ABC123',
      overdraw = false,
      note = 'Impound payment',
    })
    ```

  </Tab>
</Tabs>


## OxAccount.depositMoney

**パラメータ**

- playerId: `number`
- amount: `number`
- message?: `string`

**戻り値**

## OxAccount.deleteAccount

アカウントを削除済みとしてマークします。アクセスできなくなりますが、データベースには残ります。

```lua
account.deleteAccount()
```

**戻り値**

- `object`
  - success: `boolean`
  - message: `'something_went_wrong'`

```lua
account.getCharacterRole(id)
```

```lua
account.setCharacterRole(id, role)
```

**パラメータ**

**戻り値**

- `object`

## OxAccount.playerHasPermission

プレイヤーのアクティブなキャラクターがアカウントでアクションを実行する権限を持っているかどうかを確認します。

```lua
account.playerHasPermission(playerId, permission)
```

**パラメータ**

- playerId: `number`
- permission: `string`

**戻り値**

- `boolean`


## OxAccount.setShared

アカウントを共有として設定し、他のキャラクターに権限を割り当てられるようにします。

```lua
account.setShared()
```

**戻り値**

- `object`
  - success: `boolean`
  - message: `'update_account_error'`


## OxAccount.createInvoice

アカウントに未払いの請求書を作成します。

```lua
account.createInvoice(invoice)
```

**パラメータ**

- invoice: `object`
  - actorId?: number;
    - 請求書を作成するプレイヤーのcharId。
  - toAccount: number;
    - 請求書を受け取るアカウントのアカウントID。
  - amount: number;
  - message: string;
  - dueDate: string;

**戻り値**

- `object`
  - success: `boolean`
  - message: `'no_charId'` | `'no_permission'` | `'no_target_account'`
