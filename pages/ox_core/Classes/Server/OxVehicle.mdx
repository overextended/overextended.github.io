title: OxVehicle
---

## OxVehicle

- entity: `number`
- make: `string`
- model: `string`
- netId: `number`
- plate: `string`
- group?: `string`
- id?: `number`
- owner?: `number`
- vin?: `string`

## OxVehicle.delete

車両をデスポーンし、データベースから削除します。

```lua
vehicle.delete()
```


## OxVehicle.despawn

車両をデスポーンし、オプションでデータベースに保存します。

```lua
vehicle.despawn(save)
```

**パラメータ**

- save?: `boolean`


## OxVehicle.get

車両のメタデータから特定のキーの値を取得します。

```lua
vehicle.get(key)
```

**パラメータ**

- key: `string`

**戻り値**

- `unknown`


## OxVehicle.getCoords

車両の現在の座標を返します。

```lua
vehicle.getCoords()
```

**戻り値**

- `vector3`

## OxVehicle.getProperties

車両のプロパティオブジェクトを返します。

```lua
vehicle.getProperties()
```

**戻り値**

- `VehicleProperties`



## OxVehicle.getState

車両のステートバッグインターフェースを返します。

```lua
vehicle.getState()
```

**戻り値**

- `StateBagInterface`


## OxVehicle.getStored

車両の「保管」状態を返します。

```lua
vehicle.getStored()
```

**戻り値**

- `string?`


## OxVehicle.respawn

車両の新しいエンティティをスポーンし、既存のエンティティが存在する場合は削除し、保存されている車両プロパティを再適用します。

```lua
vehicle.respawn(coords, rotation)
```

**パラメータ**

- coords?: `vector3`
- rotation?: `vector3`


## OxVehicle.save

車両をデータベースに保存します。

```lua
vehicle.save()
```

**戻り値**

- `number`


## OxVehicle.set

車両のメタデータに値を保存します。これはデータベースに保存されます（この動作は削除される可能性があります）。

```lua
vehicle.set(key, value)
```

**パラメータ**

- key: `string`
- value: `any`


## OxVehicle.setGroup

指定されたグループ名によって「所有」されるグループ車両として車両を設定するか、省略された場合はグループを削除します。

```lua
vehicle.setGroup(groupName)
```

**パラメータ**

- groupName?: `string`


## OxVehicle.setOwner

指定されたcharIdとして車両の所有者を設定するか、省略された場合は所有者を削除します。

```lua
vehicle.setOwner(charId)
```

**パラメータ**

- charId?: `number`


## OxVehicle.setPlate

車両の登録プレートを指定された値に設定します。プレートは8文字にパディングまたはトリミングされます。

このプレートは、必ずしも車両プロパティに表示されるプレートではありません。

```lua
vehicle.setPlate(plate: string)
```


## OxVehicle.setProperties

車両に保存されているプロパティ（改造、エクストラなど）を保存し、オプションでエンティティに適用します。

```lua
vehicle.setProperties(properties, apply)
```

**パラメータ**

- properties: `VehicleProperties`
- apply?: `boolean`


## OxVehicle.setStored

車両を指定された値（例：押収、ガレージ、null）で「保管」として設定し、オプションで車両をデスポーンして保存します。

```lua
vehicle.setStored(value, despawn)
```

**パラメータ**

- value?: `string`
- despawn?: `boolean`
