## OxPlayer

- charId: `number`
- identifier: `string`
- ped: `number`
- source: `number`
- stateId: `string`
- userId: `number`
- username: `string`


## OxPlayer.addLicense

プレイヤーにライセンスを付与します。

```lua
player.addLicense(licenseName)
```

**パラメータ**

- licenseName: `string`

**戻り値**

- `boolean`


## OxPlayer.addStatus

```lua
player.addStatus(statusName, value)
```

**パラメータ**

- statusName: `string`
- value: `number`

**戻り値**

- `boolean`


## OxPlayer.createCharacter

まだキャラクターを選択していないプレイヤーのために新しいキャラクターを登録します。

```lua
player.createCharacter(data)
```

**パラメータ**

- data: `object`
  - firstName: `string`
  - lastName: `string`
  - gender: `string`
  - date: `number`

**戻り値**

- `number`
  - プレイヤーが所有するすべてのキャラクターの配列内の新しいキャラクターのインデックス。


## OxPlayer.deleteCharacter

指定されたcharIdを持つキャラクターを削除します。ただし、プレイヤーが所有しており、キャラクターを選択していない場合に限ります。

```lua
player.deleteCharacter(charId)
```

**パラメータ**

- charId: `number`

**戻り値**

- `boolean`

## OxPlayer.hasPermission

ユーザーがいずれかのグループから権限を付与されているかどうかを確認します。

```lua
player.hasPermission(permission)
```

**パラメータ**

- permission: `string`

**戻り値**

- `boolean`

**例**

```lua
-- グループに権限を追加する
Ox.SetGroupPermission('police', 1, 'handcuff', 'allow')

-- プレイヤーが「handcuff」権限を持っているかどうかを確認する
if player.hasPermission('group.police.handcuff') then
    -- 何かをする
end
```


## OxPlayer.logout

アクティブなキャラクターのデータをクリアし、有効な場合はプレイヤーをキャラクター選択に移行します。

```lua
player.logout(save, dropped)
```

**パラメータ**

- save?: `boolean`
  - アクティブなキャラクターデータをデータベースに保存するかどうか。デフォルトは`true`。
- dropped?: `boolean`
  - プレイヤーがサーバーから切断されたかどうか。


## OxPlayer.payInvoice

```lua
player.payInvoice(invoiceId)
```

**パラメータ**

- invoiceId: `number`

**戻り値**

- `object`
  - success: `boolean`
  - message?: `"no_charId"` | `"no_permission"` | `"no_balance"` | `"no_invoice"` | `"invoice_paid"` | `"insufficient_balance"`

## OxPlayer.setActiveCharacter

キャラクターをロードし、プレイヤーのアクティブなキャラクターとして設定します。charIdまたは新しいキャラクターデータを含むオブジェクトを取ります。

```lua
player.setActiveCharacter(data)
```

**パラメータ**

- data: `number` | `object`
  - firstName: `string`
  - lastName: `string`
  - gender: `string`
  - date: `number`

**戻り値**

- `object`
  - charId: `number`
  - stateId: `string`
  - firstName: `string`
  - lastName: `string`
  - x?: `number`
  - y?: `number`
  - z?: `number`
  - heading?: `number`
  - lastPlayed?: `string`
  - health?: `number`
  - armour?: `number`
  - isNew?: `boolean`


## OxPlayer.setActiveGroup

プレイヤーが属するグループをアクティブまたは「プライマリ」グループとして設定します。引数が渡されない場合、アクティブなグループは設定されません。

```lua
player.setActiveGroup(groupName?: string, temporary?: boolean)
```

**パラメータ**

- groupName: `string`
- temporary?: `boolean`
  - `true`の場合、セッション間で永続化されません。

**戻り値**

- `boolean`


## OxPlayer.setGroup

グループ内のプレイヤーのグレードを設定します。グレードが未定義または0の場合、グループから削除されます。

```lua
player.setGroup(groupName, grade)
```

**パラメータ**

- groupName: `string`
- grade?: `number`

**戻り値**

- `boolean`
