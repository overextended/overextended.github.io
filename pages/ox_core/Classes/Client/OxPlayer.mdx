## OxPlayer

- state: `StateBagInterface`
- userId: `number`
- charId?: `number`
- stateId?: `string`

## OxPlayer.addStatus

ステータスの値を増やします。

```lua
player.addStatus(statusName, value)
```

**パラメータ**
- statusName: `string`
- value: `number`

**戻り値**
- success: `boolean`

## OxPlayer.hasPermission

ユーザーがいずれかのグループから付与された権限を持っているかどうかを確認します。

```lua
player.hasPermission(permission)
```

**パラメータ**
- permission: `string`

**戻り値**
- `boolean`

**例**

```lua
-- グループに権限を追加する（サーバー側）
Ox.SetGroupPermission('police', 1, 'handcuff', 'allow')

-- プレイヤーが「handcuff」権限を持っているかどうかを確認する
if player.hasPermission('group.police.handcuff') then
    -- 何かをする
end
```

## OxPlayer.on

キーで指定されたプレイヤーのメタデータの変更をリッスンします。

```lua
player.on(key, cb(data))
```

**パラメータ**
- key: `string`
- cb: `function`

**戻り値**
- data: `unknown`
  - `key`の新しい値
