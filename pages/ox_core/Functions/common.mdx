## Ox.GetGroupPermissions

グループの `OxGroupPermissions` オブジェクトを返します

```lua
Ox.GetGroupPermissions(groupName)
```

**パラメータ**

- groupName: `string`

**戻り値**

- `OxGroupPermissions`

## Ox.GetTopVehicleStats

全体または特定のカテゴリ（陸、空、海）のトップ車両統計を含むオブジェクトを返します。

```lua
Ox.GetTopVehicleStats(category?)
```

**パラメータ**

- category?: `string`

**戻り値**

- `VehicleStats` | `{ [string]: VehicleStats }`

### VehicleStats
- `object`
  - acceleration: `number`
  - braking: `number`
  - handling: `number`
  - speed: `number`
  - traction: `number`


## Ox.GetVehicleData

すべての車両のVehicleData、またはオプションで特定のモデルまたはモデルの配列のVehicleDataを返します。

```lua
Ox.GetVehicleData(filter?)
```

**パラメータ**

- filter?: `string` | `string[]`

**戻り値**

- `{ [string]: VehicleData }` | `VehicleData`

### VehicleData
- `object`
  - class: `VehicleClasses`
  - doors: `number`
  - make: `string`
  - name: `string`
  - price: `number`
  - seats: `number`
  - type: `string`
  - weapons?: `true`

## Ox.GetVehicleNetworkType

車両モデルのnetobjタイプを返します。これは車両タイプとは異なる場合があります（例：blimpはheliとしてネットワーク化されます）。

```lua
Ox.GetVehicleNetworkType(modelName)
```

**パラメータ**

- modelName: `string`

**戻り値**

- `string`
