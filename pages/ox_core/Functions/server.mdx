## Ox.CreateAccount

指定されたcharIdまたはstateIdを持つキャラクターの新しいアカウントを作成します。

```lua
Ox.CreateAccount(ownerId, label)
```

**パラメータ**

- ownerId: `number` | `string`
- label: `string`

**戻り値**

- [`OxAccount`](/ox_core/Classes/Server/OxAccount)

## Ox.CreateGroup

指定されたデータから新しいグループを作成し、すべてのデータベース操作を内部的に処理します。

```lua
Ox.CreateGroup(data)
```

**パラメータ**

- data: `object`
  - name: `string`
  - label: `string`
  - grades: `object[]`
    - label: `string`
    - accountRole?: `string`
  - type?: `string`
  - colour?: `number`
  - hasAccount?: `boolean`

## Ox.CreateVehicle

新しい車両を作成します。割り当てられた所有者またはグループがある場合、データベースに保存されます。

```lua
Ox.CreateVehicle(data, coords, heading)
```

**パラメータ**

- data: `string` | `object`
  - model: `string`
  - owner?: `number`
  - group?: `string`
  - stored?: `string`
  - properties?: `object`
- coords?: `vector3`
- heading?: `number`

**戻り値**

- [`OxVehicle`](/ox_core/Classes/Server/OxVehicle)

## Ox.SetGroupPermission

指定されたグループと最小グレードに権限を割り当てます。これは上位のグレードに継承されます。

```lua
Ox.SetGroupPermission(groupName, grade, permission, value)
```

**パラメータ**

- groupName: `string`
- grade: `number`
- permission: `string`
- value: `allow` | `deny`

**例**

```lua
-- グループに権限を追加する（サーバー側）
Ox.SetGroupPermission('police', 1, 'handcuff', 'allow')

-- プレイヤーが「handcuff」権限を持っているかどうかを確認する
if player.hasPermission('group.police.handcuff') then
    -- 何かをする
end
```
