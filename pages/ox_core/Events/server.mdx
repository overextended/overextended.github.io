# サーバーイベント

<Callout type="warning">
これらのイベントは、アクションが発生したときにox_coreによってトリガーされます。他のスクリプトからトリガーしないでください。
</Callout>

## ox:savedPlayers

プレイヤーがデータベースに保存されたとき。
<Callout type="info">
デフォルトでは、このイベントは次の場合にトリガーされます。
* 10分ごと。
* サーバーがシャットダウンしたとき。
* [`Ox.SaveAllPlayers`](/ox_core/Functions/server#oxsaveallplayers) が呼び出されたとき。
* コマンド `saveplayers` が使用されたとき。
</Callout>
<Tabs items={['Lua', 'JS']}>
  <Tabs.Tab>
    ```lua copy
    AddEventHandler('ox:savedPlayers', function(playersSaved)
        print(playersSaved)
    end)
    ```
  </Tabs.Tab>
  <Tabs.Tab>
  ```ts copy
  on('ox:savedPlayers', (playersSaved: number) => {
    console.log(playersSaved);
  });
  ```
  </Tabs.Tab>
</Tabs>
* playersSaved: `number`
