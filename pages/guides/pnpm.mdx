# pnpm

pnpmは、高速でディスクスペース効率の良いパッケージマネージャーであり、npmやyarnの代替として機能します。

- [Node.jsがインストールされている](./nodejs)ことを確認してください。
- コマンドラインターミナル（例：ターミナル、コマンドプロンプト）を開きます。
- `npm install -g pnpm`と入力して、パッケージをグローバルにインストールします。
- `pnpm i`を使用して、pnpmで依存関係をインストールできます。

利用可能なスクリプトについては、`package.json`ファイルを参照できます。つまり、

```json
"scripts": {
  "start": "vite",
  "watch": "vite build --watch",
  "build": "tsc && vite build",
  "preview": "vite preview",
  "format": "prettier --write \"./src/**/*.{ts,tsx,css}\""
},
```

上記の例を使用すると、`pnpm build`は特定のパッケージのビルドスクリプトを実行します。
