# Git

Gitはバージョン管理システムであり、変更を効果的に追跡、マージ、元に戻すことができます。

私たちのリソースはGitHubでホストされているため、変更を追跡し、最新の状態に保つために独自のリポジトリを作成する方法を学ぶことは非常に役立ちます。私たちのリソースに独自の変更を加える予定がある場合、これにより独自の変更を保持し、データの損失を防ぐことができます。

- [Gitの設定に関するGitHubのガイド](https://docs.github.com/ja/get-started/quickstart/set-up-git#setting-up-git)を参照してください。
- [GitKraken](https://www.gitkraken.com/)のようなGit GUIクライアントを使用して、ワークフローを改善し、変更をより簡単に維持することができます。
