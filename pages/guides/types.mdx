<Steps>
    ### git、vscode、およびvscode拡張機能のセットアップ

    ### Lua型定義のダウンロード

    ```bash
    git clone https://github.com/overextended/ox_types.git`
    ```

    ### 設定の更新

    vscodeのユーザー、ワークスペース、またはフォルダーの設定を変更できます。
    クローンされたディレクトリ、または特定のサブディレクトリを指すエントリを`Lua.workspace.library`に作成します。

    ```json
    "Lua.workspace.library": [
        "F:/GitHub/ox_types/types",
        "F:/GitHub/ox_lib"
    ],
    ```

</Steps>
