# VS Code

Visual Studio Codeは、Luaを使用する際に推奨されるエディターです。

<DownloadButton link="https://code.visualstudio.com/Download" />

## 推奨拡張機能

拡張機能を使用すると、言語、デバッガー、およびその他のツールをVS Codeに追加して、開発者エクスペリエンスを向上させることができます。

### sumneko.lua

Lua Language Serverは、開発をより簡単かつ迅速にするための多数の言語機能を提供します。例：

- アノテーション
- オートコンプリート
- コードフォーマット
- 診断/警告
- 動的型チェック
- 構文チェック

### overextended.cfxlua-vscode

Lua Language Serverと連携して、Cfxの[Luaランタイム](https://github.com/citizenfx/fivem/blob/master/code/components/citizen-scripting-lua/src/LuaScriptRuntime.cpp)、[Luaダイアレクト](https://github.com/citizenfx/lua/tree/luaglm-dev/cfx)、[環境グローバル](https://github.com/citizenfx/fivem/blob/master/data/shared/citizen/scripting/lua/scheduler.lua)、およびゲームネイティブからの関数と型の宣言を提供します。
