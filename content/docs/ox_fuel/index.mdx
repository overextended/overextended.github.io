A simple fuel resource meant to replace LegacyFuel or serve as a showcase for using petrol cans as an item.

## Installation

### Install all resource dependencies. [step]

- [ox_lib](../ox_lib/)
- [ox_inventory](../ox_inventory/)

### Download a release or clone the source code. [step]

- Download the [latest release](https://github.com/overextended/ox_fuel/releases/latest).

```bash
git clone https://github.com/overextended/ox_fuel.git
```

### Install optional dependencies. [step]

These resources aren't required but provide additional functionality.

- [ox_target](../ox_target/)
