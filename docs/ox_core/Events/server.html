Ox CoreEvents

Server Events

These events are triggered by ox_core when an action occurs. You should not trigger them from other scripts.

ox:savedPlayers

When players are saved to the database.

By default this event is triggered:

  • Every 10 minutes.
  • When the server is shutdown.
  • When Ox.SaveAllPlayers is called.
  • When the command saveplayers is used.
  • playersSaved: number

ox:setActiveGroup

When the active group of a character has changed.

  • playerId: number
  • groupName: string
  • previousGroupName: string | undefined

ox:setGroup

When a character is added or removed from a group.

  • playerId: number
  • groupName: string
  • grade?: number

ox:licenseAdded

When a character receives a new license.

  • playerId: number
  • licenseName: string

ox:licenseRemoved

When a character's license is removed.

  • playerId: number
  • licenseName: string

ox:playerLogout

When a player logs out from their current character, crashes, or otherwise disconnects from the server.

  • playerId: number
  • userId: number
  • charId: number

ox:createdCharacter

When a player has created a new character.

  • playerId: number
  • userId: number
  • charId: number

ox:playerLoaded

When a player selects and loads a character.

  • playerId: number
  • userId: number
  • charId: number

ox:deletedCharacter

When a character is deleted.

  • playerId: number
  • userId: number
  • charId: number

ox:savedVehicles

When vehicles are saved to the database.

  • vehiclesSaved: number

ox:spawnedVehicle

When an OxVehicle has spawned.

  • entityId: number
  • id: number

ox:despawnVehicle

Before an OxVehicle gets despawned.

  • entityId: number
  • id: number

ox:updatedBalance

When an account's balance is updated.

  • data table
    • accountId: number
    • amount: number
    • action: 'add' | 'remove'
  • data object
    • accountId: number
    • amount: number
    • action: 'add' | 'remove'

ox:transferredMoney

When an account's money is transferred.

  • data table
    • fromAccountId: number
    • toAccountId: number
    • amount: 'add' | 'remove'
  • data object
    • fromAccountId: number
    • toAccountId: number
    • amount: 'add' | 'remove'

ox:depositedMoney

When money is deposited into an account.

  • data table
    • playerId: number
    • accountId: number
    • amount: number
  • data object
    • playerId: number
    • accountId: number
    • amount: number

ox:withdrewMoney

When money is withdrawn from an account.

  • data table
    • playerId: number
    • accountId: number
    • amount: number
  • data object
    • playerId: number
    • accountId: number
    • amount: number

ox:invoicePaid

When an invoice is paid.

  • data table
    • amount: number
    • payerId: number
    • fromAccount: number
    • toAccount: number
  • data object
    • amount: number
    • payerId: number
    • fromAccount: number
    • toAccount: number
