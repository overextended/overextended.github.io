GetSlot

Returns the specified slot data as a table.


exports.ox_inventory:GetSlot(inv, slot)
  • inv: string or number
  • slot: number

Example:

local slot = exports.ox_inventory:GetSlot(source, 1)

print(json.encode(slot, {indent=true}))
--[[
    {
        "weight": 2000,
        "name": "water",
        "metadata": [],
        "slot": 1,
        "label": "Water",
        "close": true,
        "stack": true,
        "count: 4
    }
]]
